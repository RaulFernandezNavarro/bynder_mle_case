# Where does Bynder Store Assets – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013934099-Where-does-Bynder-Store-Assets

By default, Bynder assets are located in Frankfurt, Germany. However the assets can also be hosted in other regions Bynder supports. You need to decide on the region where the data is to be hosted before your Bynder environment is configured.

Supported Global regions[](https://support.bynder.com/hc/en-us/articles/360013934099-Where-does-Bynder-Store-Assets#h_01HW67TVEGAK737ERHMM9D9HTM)

--------------------------------------------------------------------------------------------------------------------------------------------------

The below regions apply to customers who are hosted on **Global**.

*   ap-northeast-1 (S3 server in Asia Pacific, Tokyo)
    
*   eu-central-1 (S3 server in EU, Frankfurt)
    
*   us-east-1 (S3 server in US East - North Virginia)
    
*   us-west-1 (S3 server in US West - California)
    

Supported US-only regions[](https://support.bynder.com/hc/en-us/articles/360013934099-Where-does-Bynder-Store-Assets#h_01HW67TVEGECRSERG06QHGV8B3)

---------------------------------------------------------------------------------------------------------------------------------------------------

The below regions apply to customers who are hosted in **US only**.

*   us-east-2 (S3 server in US East - Ohio)
    
*   us-west-2 (S3 server in US West - Oregon)
