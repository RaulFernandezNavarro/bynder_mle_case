Gradients are a powerful and easy to use tool for ultimate customization of your colors within Studio elements. Bynder allows gradients on any element that allows color customization, and can even be used for your brand color palette(s). 

Who?[](https://support.bynder.com/hc/en-us/articles/25923547221394-How-To-Create-And-Use-Gradients-In-Studio#h_01JRD54H0EGKGDRAE5AMB1GB43)

-------------------------------------------------------------------------------------------------------------------------------------------

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Once Studio is enabled _Admins_ and _Content Designers_ can create and use the Gradients option.

Why?[](https://support.bynder.com/hc/en-us/articles/25923547221394-How-To-Create-And-Use-Gradients-In-Studio#h_01JRD54NCHJY1M7PHM8RE4HSME)

-------------------------------------------------------------------------------------------------------------------------------------------

Gradients allow the Studio user a wider range of color options. This exceptionally broadens a users creative capability, allowing for brand consistency and customization. Users can create gradients that can be applied to the brand color profile. 

How?[](https://support.bynder.com/hc/en-us/articles/25923547221394-How-To-Create-And-Use-Gradients-In-Studio#h_01JRD54T5PZ84488ZTW5AA6Q41)

### Creating A Gradient In A Design

1\. Navigate to Studio in your portal.

2\. Edit or create a design.

3\. Select any element that has a color property within the design. 

![color picker.png](https://support.bynder.com/hc/article_attachments/25923578734098)

4\. Select the color picker, and decide between the three color options:

*   **Solid color:** Solid color is the default color option within Studio. 
*   **Linear Gradient:** A gradual color progression between the selected colors on a linear, this can be horizontal, vertical, and diagonal. 

![gradient lin.png](https://support.bynder.com/hc/article_attachments/25923578735122)

*   **Radial Gradient:** A gradual color progression between the selected colors from a central focal point. The radial gradient will follow the aspect ratio of the element it was created within.

![radial.png](https://support.bynder.com/hc/article_attachments/25923547218706)

5\. Change the colors to what is needed. The first color can be selected via clicking the color picker, in combination with the hue slider or entering in the hexcode of the desired color.

*   _Optional:_ Add more colors to the gradient by creating the color stopper points along the gradient slider (not the hue slider). Users do this by clicking the gradient slider at the position they should add another color to. The color stopper points can be moved along the slider to change the gradient appearance.
    *   If only using two colors the second color along the gradient slider should be the far right color stopper point. The gradient slider by default has two color stopper points. The gradient slider goes from left to right for color order. 
        *   Add colors by selecting the correct color stopper point, and change the color to the desired shade. 

6\. _Optional_: Change the angle of the linear gradient by selecting the numerical value within the degrees section to match your visual needs. 

![angle.png](https://support.bynder.com/hc/article_attachments/25923578736018)

7\. Save or Publish the design.

### Creating A Gradient In Brand Colors

1\. Navigate to Studio.

2\. Navigate to Colors on the left-hand menu.

3\. Click Add color palette.

4\. A pop-up will appear to name the color palette. Name the color palette and press Create color palette.

5\. Open the color palette and click Add Color. 

6\. A pop-up will appear to name the gradient. Name the gradient, and click Hex code.

7\. This will open the colors modal. Use the steps above to create your brand gradient.

8\. Click create color, this will add the gradient to the brand palette. Repeat if necessary for other colors and gradients. 

_Note: Basic Studio users only have one brand palette available. To enable more please reach out to your Customer Success Contact._
