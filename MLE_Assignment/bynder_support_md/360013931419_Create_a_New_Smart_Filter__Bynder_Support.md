# Create a New Smart Filter – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013931419

Smart filters are convenient drop-down menus located at the top of the Asset Bank. Each smart filter can hold one or more metaproperties. The metaproperties in the smart filter show a count next to each option, making it easy for users to find assets quickly. To reduce the initial display, you can make [one metaproperty dependent on another.](https://support.bynder.com/hc/en-us/articles/19461824535698)

Along with the [search bar](https://support.bynder.com/hc/en-us/articles/360013931359)
, smart filters are the main method for users to [discover content in the Asset Bank.](https://support.bynder.com/hc/en-us/articles/360013870120)
 A smart filter drop-down can present a single metaproperty, like **Year**, for users to choose from. It can also include multiple metaproperties, which is beneficial for logically connected metaproperties such as Continent, Country, and City.

**Who can manage smart filters?**

Users with the following rights can create, edit, and remove smart filters:

*   [**Manage smart filters**](https://support.bynder.com/hc/en-us/articles/18433246503442-Permissions-in-Asset-Bank#:~:text=Manage%20smart%20filters%C2%A0%2D%20Allows%20users%20to%20manage%20smart%20filters.)
    

How to Create a Smart Filter[](https://support.bynder.com/hc/en-us/articles/360013931419#h_01JHNRHBZK2E20T9031WP0KD71)

-----------------------------------------------------------------------------------------------------------------------

1.  Navigate to your portal.
    
2.  Select ![settings_menu.png](https://support.bynder.com/hc/article_attachments/10641634342418) **Settings > Taxonomy > Smartfilter Management**.
    
3.  Click **+Add** in the top right corner. 
    
4.  **Enter or Select Filter Details**:
    
    1.  **Z-Index:** This number determines the filter's position from left to right at the top of the Asset Bank. Lower numbers place the filter further to the left.
        
        *   Leave space between index values (e.g., 10, 20, 30) to simplify adding new filters in the future without reordering existing ones.
            
    2.  **Use** **icons** from Font Awesome.
        
        *   To use an icon, click it on the [Font Awesome website](https://fontawesome.com/v4/icons/)
             to view the full name, then copy and paste it into the **Icon** field.![fontawesome_example](https://support.bynder.com/hc/article_attachments/10641634365586)
            
        *   Font Awesome version 5 or above icons are not supported
            
    3.  Select the [filter type.](https://support.bynder.com/hc/en-us/articles/360013931419#h_01JHNRPS8C23WMNCKXQ385WNYF)
        
    4.  Select the [filter optimization options.](https://support.bynder.com/hc/en-us/articles/360013931419#h_01JHNRTV4ZEMS0Q64SB3RNYPRY)
        
    5.  (Optional) [Customize Language](https://support.bynder.com/hc/en-us/articles/360013931419#01JHNS1VDFKS0JMNYN9711NWWR)
        .
    6.  Select **Save.** 

Filter Types[](https://support.bynder.com/hc/en-us/articles/360013931419#h_01JHNRPS8C23WMNCKXQ385WNYF)

-------------------------------------------------------------------------------------------------------

The most common type is Meta, which displays metaproperties.

1.  **Advanced**: Adds an [Advanced filter](https://support.bynder.com/hc/en-us/articles/360013931839)
     with default options. 
    
2.  **Color**: Adds a color picker filter for filtering assets by primary hex code.
    
    The color picker cannot filter color ranges.
3.  **Meta**: Displays [metaproperties in the smart filter.](https://support.bynder.com/hc/en-us/articles/19461847064466)
     
    
4.  **Tags**: Displays the top 25 most frequently [tagged](https://support.bynder.com/hc/en-us/articles/11306561722770)
     keywords.
    

Filter Optimization Options[](https://support.bynder.com/hc/en-us/articles/360013931419#h_01JHNRTV4ZEMS0Q64SB3RNYPRY)

----------------------------------------------------------------------------------------------------------------------

1.  **Lazy** **Loading**: Loads filter options incrementally as the user scrolls. Ideal for lists with 50+ options.
    
2.  **Full** **Screen**: Displays the filter dropdown in full-screen mode, covering the Asset Overview. Best for large filters.
    
3.  **Close When Selected:** This option closes the filter dropdown after each selection. It is not recommended if users often apply multiple filters.
    

### Media Without

At the top of the smartfilter, there will be an option called **Media Without** that shows any asset that currently _does not have_ that metaproperty set, meaning there is no value filled in, for that specific metaproperty.

This feature allows users to quickly locate assets that might be missing metadata.

Customize Language [](https://support.bynder.com/hc/en-us/articles/360013931419#01JHNS1VDFKS0JMNYN9711NWWR)

------------------------------------------------------------------------------------------------------------

1.  Enter the filter label in English. If your portal supports additional languages, also enter translations for those languages. **(English is the default language.)**
