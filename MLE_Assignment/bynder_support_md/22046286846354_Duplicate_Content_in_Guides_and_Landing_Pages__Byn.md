# Duplicate Content in Guides and Landing Pages – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/22046286846354-Duplicate-Content-in-Guides-and-Landing-Pages

Duplicating existing Pages in Guides or Landing Pages with all their content rather than starting from scratch can streamline your content creation and save time. When duplicating a page, you can keep the new page within the same Guide or move it to a different one. This is especially helpful when creating multiple pages or landing pages with minor variations.

### Tip

Leverage **pre-built pages** for Guides and Landing pages by duplicating and customizing them to fit your needs.

![](https://support.bynder.com/hc/article_attachments/22046288783506)

How to Duplicate A Page[](https://support.bynder.com/hc/en-us/articles/22046286846354-Duplicate-Content-in-Guides-and-Landing-Pages#h_01HF4A8EQ8V8RJAJ74VR9CEPXD)

------------------------------------------------------------------------------------------------------------------------------------------------------------------

Note: Anyone with access to the selected Guide can access this page unless specified otherwise.

1.  Navigate to the Guide that contains the page you want to replicate.
2.  If the guide isn’t in editing mode, click **Edit** **Guide**.
3.  Make sure you are on the page you want to duplicate. 
4.  Select ![](https://support.bynder.com/hc/article_attachments/22046286841618)at the bottom of the page.![](https://support.bynder.com/hc/article_attachments/22046286842642)
5.  Select the location of the duplicate page. By default, the page will be added to the Guide you are currently in. 
    
    ![](https://support.bynder.com/hc/article_attachments/22046286844690)
    
    *   (Optional) You can relocate the duplicated Page to a different Guide.
        *   If you choose this option, you can select the **Visit page** to be redirected to the duplicate page. 
            
            ![](https://support.bynder.com/hc/article_attachments/22046394155666)
            
6.  Select **Duplicate.**
