# How To Create and Manage Themes in CX For User Community – Bynder Support

Summary[](https://support.bynder.com/hc/en-us/articles/4410619103634-How-To-Create-and-Manage-Themes-in-CX-For-User-Community#h_01JNZKYBBRTXE922DNBF6TFJF2)

------------------------------------------------------------------------------------------------------------------------------------------------------------

Configuring a theme in Content Experiences For User Community (CXUC) enables you to customize your brand’s colors, typography, and the design of the top and side navigation bars for your Guides and Landing Pages.

Who?[](https://support.bynder.com/hc/en-us/articles/4410619103634-How-To-Create-and-Manage-Themes-in-CX-For-User-Community#h_01JNZM1ZJW817Y5K0REQ149RKF)

---------------------------------------------------------------------------------------------------------------------------------------------------------

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Once enabled users with the _Manage Portal Look & Feel_ permission can set a theme.

_Note: Only one theme will be available if you're using the basic package of CXUC.  Guides and Landing Pages have a default Bynder theme, which cannot be edited or removed._

Why?[](https://support.bynder.com/hc/en-us/articles/4410619103634-How-To-Create-and-Manage-Themes-in-CX-For-User-Community#h_01JNZM1CH44BGZJJD1MYMW0CQB)

---------------------------------------------------------------------------------------------------------------------------------------------------------

Applying a theme ensures content aligns with your brand identity or campaign. Once your brand colors are set, you can easily reuse them across all of the solutions.

For organizations managing multiple brands or campaigns with distinct user identities, you can create separate themes to apply unique styles to each. Users can duplicate and modify an existing theme as needed, saving time and effort.

How?[](https://support.bynder.com/hc/en-us/articles/4410619103634-How-To-Create-and-Manage-Themes-in-CX-For-User-Community#h_01J9QDDWB15K49HDVSECC1DHWV)

### Creating A New Theme

1.  Navigate to the **CX User Community** tab (this may appear under a different name).
2.  Select **Themes.** 
    
    ![](https://support.bynder.com/hc/article_attachments/22025703905298)
    
3.  Select **New Theme**.
4.  Enter a N**ame** and (optional) **Description**. 
    
    ![](https://support.bynder.com/hc/article_attachments/22025703905938)
    
5.  (Optional) Enter a HEX color code in the **Color** field to set the color for the theme thumbnail.
6.  Select **Create Theme.**

### Customizing A Theme 

1.  To customize the theme, you can select **Themes** in the left sidebar or from within your Guides or Landing Page while in **edit mode** > select  Theme, which will redirect you to the main Themes section.
2.  Click on the **Theme** you want to Customize.
    *   The default Bynder theme cannot be edited or removed.
3.  Select **Colors** in the left sidebar. 
    
    ![](https://support.bynder.com/hc/article_attachments/22025703906322)
    
4.  Select **Add Color.**
    
    ![](https://support.bynder.com/hc/article_attachments/22025703906834)
    
5.  Type in the **Color Name**
6.  Add your **HEX** **code** or select a color from the **Color Picker >** click **Save.**
7.  Select the **Fonts and Typography** to set up the fonts and typography for this template. Users can use a custom font if they desire.
    
    ![](https://support.bynder.com/hc/article_attachments/22025727281938)
    
8.  Select the ![](https://support.bynder.com/hc/article_attachments/22025703912338)under each heading option to customize. 
9.  Edit the **Name, Font Family, Font Style, and Size of the heading type,** and select the **Default Color**. ![](https://support.bynder.com/hc/article_attachments/22025703915538)
10.  Select the **Look and Feel** tab that you would like to customize. 
    *   The _Look and Feel- Guide_ tab is for your Guide header, page header, and sidebar.
    *   The _Look and Feel- Pages_ tab is for customizing your background color for your guides, homepages and landing pages.
    *   The _Look and Feel- Widgets_ tab allows users to set up predefined colors, font properties, and adjust the border radius for all widgets.
    *   You can see a preview of your customization in the right sidebar.
11.  Click **Save changes.**
12.  Select **Properties****.** This section allows you to customize. For example, if you want the main Bynder navigation bar to be visible when users are viewing the Guide or Landing Page, you can add a custom logo to the Guide header. ![](https://support.bynder.com/hc/article_attachments/22025727285138)
13.  Once you've configured your theme, you can apply it.

### Editing And Deleting A Theme 

1.  Navigate to the **Theme** page.
2.  Select the **Theme** you would like to edit.  
    *   If you would like to delete the Theme, select the ![Screenshot 2024-05-10 at 3.26.48 PM.png](https://support.bynder.com/hc/article_attachments/18857227941522)button.
3.  Make the necessary changes.
4.  Click **Save** **Changes**.
5.  The changes will automatically apply to any Guide or Landing Page that has the edited theme applied.

_Note: Please read the warning message carefully and click **Delete** only if you'd like to remove the **Theme**. This action cannot be undone._

### Duplicating a Theme

1.  Navigate to the **Theme** page.
2.  Select the **Theme** you would like to duplicate.
3.  Click **Duplicate**.
4.  The theme will be duplicated and appear on the theme overview page.
    
    ![Screenshot 2024-05-10 at 3.24.52 PM.png](https://support.bynder.com/hc/article_attachments/18857227942162)
    

### Applying A Theme

1.  **Create** a new Guide or Landing Page or **select** an existing one.
2.  Select the ![Screenshot 2024-05-10 at 3.13.13 PM.png](https://support.bynder.com/hc/article_attachments/18857196411538) button or **Edit Guide.**
3.  Under **Theme,** select the Theme you would like to apply from the dropdown option.
    *   If you need to change the Theme, select the![Screenshot 2024-05-10 at 3.18.13 PM.png](https://support.bynder.com/hc/article_attachments/18857196417042) button next to _Theme_, and you will be brought to the Theme page.
4.  Once you select the Theme, it will be applied, and the look and feel will change accordingly.

**Is there a limit to the number of colors I can add to a theme?**

No, there is no limit to the number of colors you can add to a theme.

**How do I know which theme is the default one?**

The _Default_ button will appear next to the default Portal theme for your account.

**Why is there an error icon next to my color?**

The Bynder theme will always be the default fallback option ensuring  you can always create content. The portal theme will have these as the default until you add brand colors.

**Why can I only create one Theme?**

Depending on your CXUC package, you may be restricted to one theme. Contact your Customer Success Contact to upgrade your package.
