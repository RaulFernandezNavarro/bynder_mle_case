# Understanding The Asset Report Dashboard – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/29141589258130-Understanding-The-Asset-Report-Dashboard

Summary[](https://support.bynder.com/hc/en-us/articles/29141589258130-Understanding-The-Asset-Report-Dashboard#h_01K4205HFNW2WWB3GH6FHKJXEF)

---------------------------------------------------------------------------------------------------------------------------------------------

Bynder offers extensive analytics options for our users. One such offering is the Asset Report. This dashboard is configured to assist _Standard and Advanced Analytics_ users to get an overview of their reports. 

The report provides a full, exportable list of all assets in a customer’s portal, including key metadata and operational fields. Users can filter most of the columns as well as export the results to CSV.

Who?[](https://support.bynder.com/hc/en-us/articles/29141589258130-Understanding-The-Asset-Report-Dashboard#h_01K4205HFR3QCPJ9XKS0EJX0G1)

------------------------------------------------------------------------------------------------------------------------------------------

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Once enabled users need the _View advanced reporting_ permission to access this dashboard.

Why?[](https://support.bynder.com/hc/en-us/articles/29141589258130-Understanding-The-Asset-Report-Dashboard#h_01K4205HFT629T2GPB2JENMWFB)

------------------------------------------------------------------------------------------------------------------------------------------

It's important for users to be able to understand which Assets are currently part of their DAM, and what their accompanying taxonomy data is. The _Assets Report Dashboard_ empowers users a self-service option to get an overview of all assets, allowing for bulk reviews, prepare for migrations, and easily accomplish internal compliance audits.

How?[](https://support.bynder.com/hc/en-us/articles/29141589258130-Understanding-The-Asset-Report-Dashboard#h_01K4205HFVMKVTFHQ378ZCWVNT)

------------------------------------------------------------------------------------------------------------------------------------------

Accessing the Asset Report dashboard is easy. Simply click the settings wheel, scroll down to **Reporting** then select **Analytics.** From here users can select the **Asset Report Dashboard** in the lefthand side menu.

![asset report.png](https://support.bynder.com/hc/article_attachments/29141561973010)

Understanding The Asset Report Dashboard[](https://support.bynder.com/hc/en-us/articles/29141589258130-Understanding-The-Asset-Report-Dashboard#h_01K4205HFY9DSZHX7DR6E5MH9X)

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

The Asset Report provides information on an individual asset level. Users have a wide range of filters available, corresponding to the columns available in the table.  Users can sort each column, and all data can be exported to a CSV file.

![assetreportvisual.png](https://support.bynder.com/hc/article_attachments/29141589256722)

The report allows for 500 rows of data to be seen in the UI. 

_Note:The report cannot be sorted by asset metadata. The entire media report needs to be exported and then filtered further in a data viewing software._
