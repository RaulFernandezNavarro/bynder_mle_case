# Understanding The Taxonomy Dashboards – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/29141834794386-Understanding-The-Taxonomy-Dashboards

Summary[](https://support.bynder.com/hc/en-us/articles/29141834794386-Understanding-The-Taxonomy-Dashboards#h_01K421J8ZDNHVVWT2YVGZHKYKG)

------------------------------------------------------------------------------------------------------------------------------------------

Bynder offers extensive analytics options for our users. The Taxonomy Dashboards: _**Metaproperties**_ and _**Automation Dependencies**_, provide a self-serve, comprehensive view of a portal’s entire taxonomy structure. These dashboards replace the need for time-consuming, ad-hoc exports by making this critical data available instantly in Analytics.

Who?[](https://support.bynder.com/hc/en-us/articles/29141834794386-Understanding-The-Taxonomy-Dashboards#h_01K421J8ZHTV7JFZD5WAQ7GCCJ)

---------------------------------------------------------------------------------------------------------------------------------------

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Once _Standard_ or _Advanced Analytics_ is enabled users need the _View advanced reporting_ permission to access these dashboards.

Why?[](https://support.bynder.com/hc/en-us/articles/29141834794386-Understanding-The-Taxonomy-Dashboards#h_01K421J8ZKS7S495Q5VRCMZ229)

---------------------------------------------------------------------------------------------------------------------------------------

It's important for users to be able to understand how the Taxonomy is structured. This empowers users easily clean up unused or duplicated metadata, prepare for portal restructuring, validate metaproperty dependencies, and build scripts for taxonomy updates. Use The Taxonomy Dashboards for proactive DAM maintenance, reduce clutter, and ensure your taxonomy stays aligned with business needs and expectations.

How?[](https://support.bynder.com/hc/en-us/articles/29141834794386-Understanding-The-Taxonomy-Dashboards#h_01K421J8ZMMXDYRZN6QGXKERVE)

---------------------------------------------------------------------------------------------------------------------------------------

Accessing the Taxonomy Report dashboard is easy. Simply click the settings wheel, scroll down to **Reporting** then select **Analytics.** From here users can select the dashboard in the left-hand side menu.

![](https://support.bynder.com/hc/article_attachments/29142177020050)

Understanding The Taxonomy Report Dashboards[](https://support.bynder.com/hc/en-us/articles/29141834794386-Understanding-The-Taxonomy-Dashboards#h_01K421J8ZR6XQ99TG4C1DEZM2A)

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

The Taxonomy Report provides two dashboards, Metaproperties and Automations. For both, users can sort on each column and all data can be exported to CSV.

The Metaproperties dashboard provides an overview of all metaproperties and their accompanying Metaproperty Options. Each row represents one Metaproperty Option.

![](https://support.bynder.com/hc/article_attachments/29142177022866)

The Automation dashboard provides an overview of all automations currently set up in the DAM. Each row represents one automation.

![](https://support.bynder.com/hc/article_attachments/29142163112594)
