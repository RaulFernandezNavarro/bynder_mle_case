# Understanding The AI Search Dashboard In Advanced Analytics – Bynder Support

Summary[](https://support.bynder.com/hc/en-us/articles/25834920685714-Understanding-The-AI-Search-Dashboard-In-Advanced-Analytics#h_01JR303858N63CP7Z9R80PGZ9E)

----------------------------------------------------------------------------------------------------------------------------------------------------------------

Bynder offers extensive analytics options for our users. One such offering is the AI Search Dashboard. This dashboard is configured to assist _Advanced Analytics_ users to get in-depth information on their users AI-Search data. This information can be provisioned with exceptional granularity as with all dashboards to allow maximum usefulness and customization of the data. Users can get data by search type, user group, and individual users to identify top performers and areas for improvement—helping you maximize AI-driven content discovery.

Who?[](https://support.bynder.com/hc/en-us/articles/25834920685714-Understanding-The-AI-Search-Dashboard-In-Advanced-Analytics#h_01JR309BKFF5JJMV0WZFH0S3EZ)

-------------------------------------------------------------------------------------------------------------------------------------------------------------

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Once _Advanced Analytics_ is enabled users need the _View advanced reporting_ permission to access this dashboard.

Why?[](https://support.bynder.com/hc/en-us/articles/25834920685714-Understanding-The-AI-Search-Dashboard-In-Advanced-Analytics#h_01JR30B5FCN3E9T4GARSV2FAS8)

-------------------------------------------------------------------------------------------------------------------------------------------------------------

It's important for Bynder Admins to be able to see their return on investment in their Bynder enabled AI offerings. Using the AI-Search dashboard Bynder Admins can assess what users have adopted the AI features enabled in their portals, and how often they are leveraging these tools within their portals. This is an excellent way to see how often these tools are being used, and what users may require more training for Bynder's AI features. 

How?[](https://support.bynder.com/hc/en-us/articles/25834920685714-Understanding-The-AI-Search-Dashboard-In-Advanced-Analytics#h_01JR30HXC0AYRX2193DHSS1G99)

The AI-Search dashboard is incredibly granular. Users can decide on several levels how they wish to see the data.

**Time Frame:** Time Frame is delineated into three sections, the date-filter selector section, the numerical amount wanted, and the time-filter selector section.

|     |     |
| --- | --- |
| **Date Filter Options:** | **Time Filter Options:** |
| *   is in the last | *   seconds |
| *   is on the day | *   minutes |
| *   is in range | *   hours |
| *   is before | *   days |
| *   is on or after | *   weeks |
| *   is in the year | *   months |
| *   is in the month | *   quarters |
| *   is this | *   years |
| *   is next | *   complete seconds |
| *   is previous | *   complete minutes |
| *   is | *   complete hours |
| *   is null | *   complete days |
| *   is anytime | *   complete weeks |
| *   is not null | *   complete quarters |
| *   matches a user attribute | *   complete years |
| *   matches(advanced) |     |

For example, if you wanted to select data that _is in the last_ | _2_ | _quarters_ that is possible.

![Screenshot 2025-04-05 at 15.21.12.png](https://support.bynder.com/hc/article_attachments/25834937340562)

**Date Granularity:** Decide how you want the data presented between _Daily, Weekly, Monthly, Quarterly_ and _Yearly._

![Screenshot 2025-04-05 at 15.23.20.png](https://support.bynder.com/hc/article_attachments/25835373278354)

**AI Search Type:** Decide what AI feature you would like to pull the data for, you may only pull data for the AI options that are enabled in your portal. The options are: _Search by Image, Search by URL, Natural Language,_ and _Show Similar._

![Screenshot 2025-08-25 at 10.48.19.png](https://support.bynder.com/hc/article_attachments/28990199533970)

_Optional data refinement options are:_

**User Profile:** Select the profile type you wish to search, such as _Admin,_ or _Removed Users._

**Group Name:** Select the group or department you wish to search, such as _Marketing,_ or _IT._

**User ID:** Select the data based on a specific user ID preconfigured in your portal. 

**User Name:** Select the user you wish to pull the data for. 

Once you have decided what data you would like to see in the dashboard you can see the data for the following:

![Screenshot 2025-04-05 at 15.36.33.png](https://support.bynder.com/hc/article_attachments/25835373278994)

**AI Searches:** Total number of AI searches.

**Searches By Image:** Total number of searches by image.

**Searches By URL:** Total number of searches by URL.

**Natural Language:** Total number of Natural Language searches.

**Show Similar:** Total number of Show Similar searches.

**Users Searching:** Unique number of users using AI search.

Next users will see the _AI Searches by Type over Time_ chart. This is a breakdown of AI searches through time.

![Screenshot 2025-04-05 at 15.41.18.png](https://support.bynder.com/hc/article_attachments/25835373279122)

### Download AI Search Dashboard Data

Downloading the dashboard data is a snap! Simply click the ![Screenshot 2025-04-05 at 15.49.09.png](https://support.bynder.com/hc/article_attachments/25835373280402)button. Once the menu opens click **Download.**

**![Screenshot 2025-04-05 at 15.48.10.png](https://support.bynder.com/hc/article_attachments/25835411216146)**

This will open a pop-up that allows you to configure the formatting of your downloaded data. You can view the data by pressing _Open in Browser_ or you can download by pressing _Download._

![Screenshot 2025-04-05 at 15.49.50.png](https://support.bynder.com/hc/article_attachments/25835411216530)

Bynder will generate your report and download the document to your device.
