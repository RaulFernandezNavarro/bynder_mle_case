# How To Configure The Download Form For Asset Intended Use – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/19269116574226

Summary[](https://support.bynder.com/hc/en-us/articles/19269116574226#h_01JXQ5TPAMEC2357GV1XKRZNGB)

----------------------------------------------------------------------------------------------------

By activating a download form, Admins can monitor how often users download assets and the reasons behind their downloads. Users will be required to specify their purpose for downloading, but this is not a download request form. The collected data from the form will be visible in the **Asset Intended Use Dashboard,** part of Standard and Advanced Analytics only.

Who?[](https://support.bynder.com/hc/en-us/articles/19269116574226#h_01JXQ6YYZX5V8HDEPF6G85EMAT)

-------------------------------------------------------------------------------------------------

This feature/solution can only be enabled by your Customer Success Contact. 

Users with the _Manage Portal Settings_ permission can use this feature once enabled.

Why?[](https://support.bynder.com/hc/en-us/articles/19269116574226#h_01JXQ6Z1SS93C3VVJ9EFV714MS)

-------------------------------------------------------------------------------------------------

The data collected from the form offers valuable insights into user actions, enabling you to showcase the asset's worth and enhance content creation strategies based on download purposes like marketing or internal needs. Additionally the form can be customized to the companies specific needs and allow for refined data collection. The download form will only appear for Bynder users who are logged in, and will not appear if an asset is downloaded via API or UCV.

How?[](https://support.bynder.com/hc/en-us/articles/19269116574226#h_01JXQ6Z5NQRBGDNZ9D0ZNW1J53)

-------------------------------------------------------------------------------------------------

### Creating And Enabling A Download Request Form

### Things to Note:

*   The form needs the included default intro question, and at least one reason for download in order to be enabled.
*   There is a 40-character limit to each value
*   Users can create up to 100 reasons for download to be displayed. However, the end user can only select up to 10 reasons at a time.
*   You cannot have two of the same value.
*   You cannot leave a value blank.
*   You can disable or delete values but cannot edit a previously created custom value.
    *   If you want to change the value, delete it, and create a new one.

1.  Navigate to your Portal> **Advanced** **Settings** > **Portal** **Settings**.
2.  Under **General** **Settings,** select **Download** **Form.** 
    
    _Note: The intro question, “How do you intend to use this asset?” **cannot** be edited but can be translated into multiple languages._  
    
    3.  *   _The introductory question can be translated into another language with a maximum of 186 characters._

**![](https://support.bynder.com/hc/article_attachments/19269116545938)**

3.  Select the **Picklist Type** from the dropdown menu: **Single** **Select** or **Multiselect** are possible. ![](https://support.bynder.com/hc/article_attachments/19269116549010)
4.  The first two options are default options and **cannot** be edited; however, they can be disabled or translated if you select the ![](https://support.bynder.com/hc/article_attachments/19269157040402) next to the selected option to disable or enable and select the![](https://support.bynder.com/hc/article_attachments/19269157043218) button to add translations.![](https://support.bynder.com/hc/article_attachments/19269157043986)
5.   To add additional options, click **\+ Add Value** and add your value to the cell. 
6.  Click **Save** **Changes**.
7.  Select the ![](https://support.bynder.com/hc/article_attachments/19269116559634)to enable the form.
8.   Select the![](https://support.bynder.com/hc/article_attachments/19269116561426) button next to the option to reorder values.
9.  Click **Save Changes,** the form is now active and can be used.

### Disabling Or Deleting The Form

1.  Navigate to your Portal> **Advanced** **Settings** > **Portal** **Settings**.
2.  Under **General** **Settings,** select **Download** **Form.**
3.  To disable or delete a custom value select the ![](https://support.bynder.com/hc/article_attachments/19269116565138)to off to disable or select the ![](https://support.bynder.com/hc/article_attachments/19269157059858) button to delete.

*   Even if a custom value is removed or disabled, the previous data will still appear in the dashboard.
*   If you delete and replace a custom value with the same value, the data will be combined with the old value of the same name.
*   If you disable all custom values, the form will be removed.

FAQs[](https://support.bynder.com/hc/en-us/articles/19269116574226#h_01HZ7JAQF1799SY3W7X2KFBW2X)

-------------------------------------------------------------------------------------------------

**If I disable the form, will my data still appear?**

Even if the form is disabled, the historical data will still appear in the Asset Intended Use dashboard.

**Can I create an open text field as an option?**

No, currently you cannot create an open text field as an option.

**Will the form appear if I have an Open Asset Bank?**

Users will see the download form as if they were logged in. If you have an open asset bank and a person who is **not** **logged** in wants to download an asset, the form **will** **appear** for them, and they will need to provide a reason for download before they can do so. However, in the asset intended use dashboard, individual users will not be tracked. All data collected from those users will be stored under the Open Asset Bank User ID.

**Can I configure the form to show only to a specific group of users or assets?**

No, this is not currently possible.

**Why can't I click save after creating my form?**

There is probably an empty cell left in the form that needs to be removed.
