# Storage Dashboard in Advanced Analytics – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/20094046094354-Storage-Dashboard-in-Advanced-Analytics

The Storage Dashboard in [Advanced Analytics](https://support.bynder.com/hc/en-us/articles/14829008886930-Advanced-Analytics)
 provides detailed insights into the storage usage within your portal. This tool is essential for monitoring and managing the space allocated to assets and workflows. The information provided will help provide trends in storage usage over time to anticipate future needs and optimize storage allocation. Based on this data and data from the **Asset Bank** dashboard you can free up some storage or clean up outdated assets to optimize space.

![Screenshot 2023-11-30 at 3.21.30 PM.png](https://support.bynder.com/hc/article_attachments/20094046091538)

**Who can view analytics reports?**

Users with the following permissions can access and download reports:

*   [**View advanced reporting**](https://support.bynder.com/hc/en-us/articles/18433246503442-Permissions-in-Asset-Bank#:~:text=View%20advanced%20reporting%20%2D%20Allows%20users%20to%20access%20Bynder%20Analytics.)
    

Total Storage[](https://support.bynder.com/hc/en-us/articles/20094046094354-Storage-Dashboard-in-Advanced-Analytics#h_01J2HCC0535JYKN2MCZD8BKQB2)

--------------------------------------------------------------------------------------------------------------------------------------------------

Displays the total amount of storage used in the portal, measured in gigabytes (GB).

*   [Assets in the Asset Bank](https://support.bynder.com/hc/en-us/articles/20094046094354-Storage-Dashboard-in-Advanced-Analytics#h_01J2HCC053KYNF4JWTNR3BV8WC)
     (including derivatives)

     (including asset previews)

    *   Excludes: Expired and revoked Bynder Express **transfers** from the calculation.

Provides a breakdown of the storage usage by each solution within the selected timeframe.

Shows the total storage used specifically within the Asset Bank during the selected time frame.

Asset Bank Storage by Original vs Derivatives[](https://support.bynder.com/hc/en-us/articles/20094046094354-Storage-Dashboard-in-Advanced-Analytics#h_01J2HCC0535T1Y7HVMG6EBECMJ)
