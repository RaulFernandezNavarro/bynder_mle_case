# Keyboard Shortcuts in Studio – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/16714804240914

Using keyboard shortcuts in Studio is a fast and efficient way to move around the platform. They help you complete tasks and use features easily, ultimately saving time and boosting productivity.

The menu of all keyboard shortcuts can be found on the screen in the bottom-right corner of the design canvas. Alternatively, use the **F1** key to open the menu

### Note

Depending on the supported browser you are using, you may need to enable privacy settings to use certain shortcuts. We recommend that you reference your supported browser's help center for further instructions.

Due to key mapping, keyboard shortcuts may differ when using a Windows (PC) keyboard on a Mac computer. The primary key used in combination with other keys often changes. Here are the substitutions for the primary key on the Windows keyboard and their equivalent on MacOS.

|     |     |
| --- | --- |
| **MacOS** | **Windows** |
| _Command ⌘  `Cmd`_ | Windows / Start |
| ⌥ `Option` | Alt `Alt` |
| ⌃ `Control` | Control `Ctrl` |
