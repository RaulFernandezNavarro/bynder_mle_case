------------------------------------------------------------------------------------------------------------------------------------------

Custom links allow you to create branded, shareable URLs, replacing long, complex links with short, on-brand vanity URLs. This article will guide you through the process of creating and managing custom links with Bynder's CX User Community feature.

Who?[](https://support.bynder.com/hc/en-us/articles/21861490210450-How-To-Create-And-Manage-Custom-Links#h_01JNR8YCQJRS7TMK0APVHWQCSG)

---------------------------------------------------------------------------------------------------------------------------------------

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Once enabled users must have at least one of the custom links specific permission to use this feature. The _view custom links_ permission allows users to view custom links shared with them. _Create custom links_ allows users to have the same permissions as _view custom links_ as well as be able to create custom links and edit those links that belong to them. _Manage custom links_ can do everything the previous permissions can as well as edit and delete all custom links. 

Why?[](https://support.bynder.com/hc/en-us/articles/21861490210450-How-To-Create-And-Manage-Custom-Links#h_01JNR94365G0T555VAAPB2FPJM)

---------------------------------------------------------------------------------------------------------------------------------------

Custom links with Bynder are incredibly versatile. Custom links can include features like password protection, start and expiration dates, and customizable QR codes, enhancing both user experience and content distribution. They can be used for public campaign landing pages, sharing collections, and sharing internal Bynder portal sections or sharing the portal's Bynder Express link. Custom links can even have language codes for localized content distribution. 

_Note: If you don't have these permissions enabled please contact your Bynder Admin._ 

How?[](https://support.bynder.com/hc/en-us/articles/21861490210450-How-To-Create-And-Manage-Custom-Links#h_01J9PNR53XZ9FF8N4HT2ZQ0RT4)

### Creating A New Custom Link

1.  Navigate to your **Portal Settings > Custom Links.** ![Screenshot 2025-01-16 at 6.57.16 PM.png](https://support.bynder.com/hc/article_attachments/23985395017874)![Screenshot 2024-11-19 at 12.16.59 PM.png](https://support.bynder.com/hc/article_attachments/22736294881554)
2.  Click the ➕ **New** **link** button.![Screenshot 2024-11-19 at 11.31.22 AM.png](https://support.bynder.com/hc/article_attachments/22736294885522)
3.  In the **Link** tab enter the **Destination field:** Enter or paste the _**entire**_ **URL** of the page you want to direct to. 
4.  Add a **Title** for the **custom link**. Once you enter a title, it will automatically be included in the custom link after the **"/by/."**
5.  You can modify the custom link name directly in the **custom** **link** field.  
6.  (Optional) Select from the available customizations (duration, password, or QR code).
7.  Once you've made your selections click **Save** to create your custom link.
8.  The link will be automatically copied to your clipboard, and ready to use!

### How To Set Duration

1.  Navigate to the **Duration** **tab** and enable the **Set** **Duration** option to set the link's start and end date.
2.  Define the link's start and end dates in the **Start** **date** and **End** **date** fields. Links will _not_ work outside of these defined dates.
3.  You can modify the duration after setting it.

_Note: To disable **Set Duration** simply toggle the button to the off position, and the link will remain active indefinitely._

![](https://support.bynder.com/hc/article_attachments/21861584167698)

### How To Password Protect Custom Links

1.  Navigate to the **Password** tab and enable the **Set Password option**. 
2.  The password _must_ be at least six characters long.
3.  Enter the password in the **Enter** **password** and **Re-enter password** fields.

_Note: To disable **Password Protection** simply toggle the button to the off position, the link will then become public._

![](https://support.bynder.com/hc/article_attachments/21861584168978)

### How To Create A QR Code For Custom Links

1.  Navigate to the **QR** **code** tab.
2.  _(Optional)_ Enable the **Logo** option to display your account logo at the center of the QR code, or  click **Select** **logo** to choose a different image for the QR code image.
3.  _(Optional)_ Use the **Foreground** **color** and **Background** **color** fields to pick colors with the color picker or by entering a hex code.![](https://support.bynder.com/hc/article_attachments/21861584171026)

### How To Edit Custom Links

1.  Navigate to your portal > **CX for User Community > Custom Links.**
2.  Select the ![](https://support.bynder.com/hc/article_attachments/21861584197906) button next to the custom link you want to edit, click **Edit**.
3.  Make the necessary changes in the available tabs.
4.  Select **Update** to save the changes.

### ![](https://support.bynder.com/hc/article_attachments/21861584198930)How to Delete Custom Links

1.  Navigate from your portal to **CX for User Community > Custom Links.**
2.  Select the ![](https://support.bynder.com/hc/article_attachments/21861584197906) button next to the custom link you want to remove and click **Delete**.
3.  Read the warning message, and click **Yes**, **delete** to _permanently_ remove the custom link and all associated analytics.

**Will I see any information on impressions or visits?**  
Yes, we have analytics to see the number of clicks on the custom link from the custom link overview page.

**How do I change the password on a custom link?**

Once a password has been added to a custom link, it can't be edited directly. To change the password, users must disable the password protection, then re-enable it with the updated password. This allows users to update access credentials while maintaining control over who can view the content.

**What happens if I delete a custom link? Will it redirect to the original URL?**

If you delete a custom link and someone tries to access it, they will receive a _Access Restricted Content_ error message. It will not redirect to the destination URL.

**Where can I see the analytics on custom links?**

In the custom links overview page, where you'll be able to see all the data relating to your custom links.
