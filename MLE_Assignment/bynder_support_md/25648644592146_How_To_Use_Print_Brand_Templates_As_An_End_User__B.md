# How To Use Print Brand Templates As An End User – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/25648644592146-How-To-Use-Print-Brand-Templates-As-An-End-User

Summary[](https://support.bynder.com/hc/en-us/articles/25648644592146-How-To-Use-Print-Brand-Templates-As-An-End-User#h_01JQEAMQ06ER00H23SF5KC3TC6)

----------------------------------------------------------------------------------------------------------------------------------------------------

Print Brand Templates allows users to create branded print materials. End users can customize premade templates based on the preconfigured edibility of the design uploaded by the Admin.

Who?[](https://support.bynder.com/hc/en-us/articles/25648644592146-How-To-Use-Print-Brand-Templates-As-An-End-User#h_01JQEAT1CPYD4AGFYCA7VQY3NP)

-------------------------------------------------------------------------------------------------------------------------------------------------

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Once _Print Brand Templates_ and _Asset Workflow_ are enabled users can create assets based on the  pre-existing templates.

Why?[](https://support.bynder.com/hc/en-us/articles/25648644592146-How-To-Use-Print-Brand-Templates-As-An-End-User#h_01JQEB21M3GKAZV4V522DZBVQK)

-------------------------------------------------------------------------------------------------------------------------------------------------

Bynder wants to ensure brand consistency for our users. We do this by allowing Print Brand Template (PBT) End Users to customize and create print assets based on the preconfigured editable aspects of the uploaded template(s). End users can only customize the sections of a template that have been predetermined by the Admin. Once customized the individual portals flow will determine where the print asset goes. This can be the Waiting Room, the Asset Bank, or can be downloaded directly from the workflow.

How?[](https://support.bynder.com/hc/en-us/articles/25648644592146-How-To-Use-Print-Brand-Templates-As-An-End-User#h_01JQEB9ND1X4R2KDCF41VR17F0)

-------------------------------------------------------------------------------------------------------------------------------------------------

1\. Navigate to the **Asset Workflow** (this can look different in customized portals, ask your Bynder Admin.).

2\. Select the **Create New** button and select from the available job presets from the menu.

![Screenshot 2025-03-28 at 13.10.46.png](https://support.bynder.com/hc/article_attachments/25648644584082)

3\. The workflow job creation pop-up will appear, users will fill in the **name,** we recommend clear and concise naming conventions. Some portals will ask for _description, accountable, and deadline,_ but the default is that the only requirement is the name. After naming this project press the **add** button.

![Screenshot 2025-03-28 at 13.27.37.png](https://support.bynder.com/hc/article_attachments/25648644584594)

4\. The job will now be created based off of the stages decided by the Admin. The first stage is by default the creation stage, where users will begin creation by pressing the **select a template** button.

![Screenshot 2025-03-28 at 13.33.17.png](https://support.bynder.com/hc/article_attachments/25648644585746)

5\. A pop up will appear showing the templates that are available for the user to customize. **Select** the template needed for this job.

![Screenshot 2025-03-28 at 13.37.49.png](https://support.bynder.com/hc/article_attachments/25648644586770)

6\. Once the template loads users will see a thumbnail of the design on the lefthand side, and the workspace for editing the template will be to the right. Users will **click on the editable aspects** of the template and change the content based on their needs. 

![Screenshot 2025-03-28 at 13.40.03.png](https://support.bynder.com/hc/article_attachments/25648650862482)

_Note: Editing images can be done using the **editable images** toolbar,![Screenshot 2025-03-28 at 14.07.29.png](https://support.bynder.com/hc/article_attachments/25648650863122)_

_or by selecting the image the user wishes to edit, and then using the **extended image toolbar** ![Screenshot 2025-03-28 at 14.07.37.png](https://support.bynder.com/hc/article_attachments/25648644587922) to make the necessary changes._

7\. Once end users are done editing the fields, they will **click the Save & continue** button.

*   The design will then go to the creation stage of the workflow. 
    *   If approvals are required, users will manually push the asset to the approval stage.
*   Some portals will have the _download preview_ button, which allows for a low resolution download of the asset.
*   You can have more than one template in the creation stage of the job.

8\. Once users are back in the stages section of the job they can _download_ the asset if possible, or click **submit to next stage** for the approval process(if enabled).

![Screenshot 2025-03-28 at 13.49.44.png](https://support.bynder.com/hc/article_attachments/25648644589074) 

9\. Once approved users will receive an email notification to their inbox. From here they can **download the asset** two ways. _This could be disabled in some end user portals, and the Admin will complete this stage._

*   Users will **click the three dots button** on the asset thumbnail and select the download option that suits the project needs best.
*   Users can click on the asset thumbnail, and can then click the **Select file to download** dropdown, and select the download option that suits the project needs best.

10\. Lastly, users will click the **Finish job & submit to to Asset bank** button.

*   This button can also be titled **Finish job,** the asset does not go to the asset bank or the waiting room, and stays within the workflow.
*   Users can go into the _complete jobs_ section of _Asset Workflow_ to download the asset at a later time.

11\. A pop-up will appear asking if the user is sure they want to add the asset to the DAM. **Click Yes.**

![Screenshot 2025-03-28 at 14.00.23.png](https://support.bynder.com/hc/article_attachments/25648644589330)

12\. A pop-up will appear asking the user to select which assets they would like to add to the DAM. **Select** the necessary assets, then **click Export.** The assets are then sent to the A_sset Bank_ or _Waiting Room,_ depending on the Admins workflow setup. 

**![Screenshot 2025-03-28 at 14.02.14.png](https://support.bynder.com/hc/article_attachments/25648650866962)**

_Note: There is the option to Select all assets if there are several being uploaded. This pop-up can be disabled by the Admin and never seen by the end user._

**Where do I find a job that I have already started?**

Go to the campaign the job exists within to find the job. End Users by default will only see jobs they have created themselves.

**Can I edit an asset I've already created?**

Yes, if the user is still within the creation stage. From the creation stage users will click asset thumbnail. This will take you to the larger version of the asset, and an edit button will be at the top of the page.
