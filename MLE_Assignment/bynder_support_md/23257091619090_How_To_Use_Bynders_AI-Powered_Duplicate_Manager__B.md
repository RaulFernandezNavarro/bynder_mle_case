# How To Use Bynder's AI-Powered Duplicate Manager – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/23257091619090

Summary[](https://support.bynder.com/hc/en-us/articles/23257091619090#h_01JVMC3ZCDD3M9S3MDRRY1M4KB)

----------------------------------------------------------------------------------------------------

The AI-Powered Duplicate Manager is your ultimate tool for maintaining a clean and efficient Asset Bank. This feature streamlines asset management by detecting and managing duplicate files across all Bynder-supported file types. Bynder has two options for Duplicate Management, _exact duplicates_ and _similar images,_ which are separated in the duplicate manager screen to allow for easier viewing of the assets. By identifying duplicates, active or archived, exact or similar, you can ensure that your DAM remains organized, up-to-date, and free of redundancy, enhancing both operational efficiency and the overall user experience.

Who?[](https://support.bynder.com/hc/en-us/articles/23257091619090#h_01JVP1MVVP2KAZ4RW17HBMXP4S)

-------------------------------------------------------------------------------------------------

This feature/solution can only be enabled by your Customer Success Contact. 

Once AI-Powered Solutions are enabled, users with the _delete assets_ permission can use Bynder's Duplicate Manager.

Why?[](https://support.bynder.com/hc/en-us/articles/23257091619090#h_01JVV98VVFYKXVNZKHQ21P6ZHY)

-------------------------------------------------------------------------------------------------

Managing your assets is a key principal within your DAM. Using Bynder's AI-Powered Duplicate Manager allows users to quickly and effectively find assets that are exact or similar. Removing assets that do not need to be in the DAM saves the portal users time, and saves on storage costs. Duplicate Manager is the perfect way to simplify this process. 

How?[](https://support.bynder.com/hc/en-us/articles/23257091619090#h_01JHGJ3VJ1526T8CKCFTF0AH34)

-------------------------------------------------------------------------------------------------

1.  Navigate to your portal and select the **Asset** tab.
    *   (Optional) Use smartfilters to refine your search in Duplicate manager. You must apply the filters before entering the duplicate manager. 
2.  Click on **Duplicate Manager** in the top-right corner.![dm1.png](https://support.bynder.com/hc/article_attachments/24122260269074)
3.  The Duplicate Manager provides an overview of duplicate assets across all file types.
4.  Click on a group to view the duplicates of a specific asset.
5.  Users can then decide to see the _Exact duplicates_, or select the _Similar images_ tab and see the images that are a close match to the selected asset. Please note that _Similar images_ is only for image file types.![Screenshot 2025-05-19 at 15.28.01.png](https://support.bynder.com/hc/article_attachments/24122260271762) ![image-20250520-092826.png](https://support.bynder.com/hc/article_attachments/26877418762642)
6.  Choose an asset to keep by selecting **Accept**.  ![screenshot_2025-01-21_at_18.44.42.png](https://support.bynder.com/hc/article_attachments/24122260272530)
7.  Once you select accept the asset will no longer appear in the duplicate overview.
    *   (Optional) To recover an asset, select the **Accepted** tab and select the asset you want to recover. Select **Recover.**![screenshot_2025-01-21_at_18.48.26.png](https://support.bynder.com/hc/article_attachments/24122237758226)
8.  Remove unwanted duplicates from your DAM by selecting **Delete**.
    *   If you accidentally delete a duplicate, you can recover it from your trash. Assets will remain in the trash section of your portal for 30 days. ![screenshot_2025-01-21_at_18.45.48.png](https://support.bynder.com/hc/article_attachments/24122237758610)

FAQs[](https://support.bynder.com/hc/en-us/articles/23257091619090#h_01JHGJ3VJ1JZE4TSX3VN5MVJD3)

-------------------------------------------------------------------------------------------------

**Will the duplicate manager respect visibility permissions?**

Yes, visibility restrictions are respected. 

**Will an asset be marked as duplicate even with a different file name?**

Yes, identical assets are marked as duplicates regardless of file name differences.

**Will archived assets appear in the Duplicate Manager view?**

Archived duplicates are used within the Duplicate Manager.

_Note: You will need the appropriate permissions to view archived assets._ 

**What if the asset has different metadata?**

Embedded metadata differences (e.g., GPS coordinates) can prevent assets from being flagged as duplicates.

**What happens to accepted assets?**

Accepted assets are not deleted but are hidden from the **Exact Duplicates Overview**. They can be accessed anytime via the **Accepted** tab in the Duplicate Manager.

**Can I recover accepted assets?**

Yes, accepted duplicates can be recovered. Once recovered, they will reappear in the overview. You can recover assets individually or in bulk.
