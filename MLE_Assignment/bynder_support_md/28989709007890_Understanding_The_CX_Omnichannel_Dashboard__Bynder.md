# Understanding The CX Omnichannel Dashboard – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/28989709007890-Understanding-The-CX-Omnichannel-Dashboard

Summary[](https://support.bynder.com/hc/en-us/articles/28989709007890-Understanding-The-CX-Omnichannel-Dashboard#h_01K3G5JENFMPSMTCQ3VAPXMA38)

-----------------------------------------------------------------------------------------------------------------------------------------------

Bynder offers extensive analytics options for our users. One such offering is the CX Omnichannel Dashboard. This dashboard is configured to assist _Advanced Analytics_ users to get in-depth information on their users CX Omnichannel data. This information can be provisioned with exceptional granularity, as with all dashboards, to allow maximum usefulness and customization of the data. Organizations can optimize asset delivery, maintain brand consistency, and track content engagement trends by analyzing these metrics.

Who?[](https://support.bynder.com/hc/en-us/articles/28989709007890-Understanding-The-CX-Omnichannel-Dashboard#h_01K3G5RJ4RJWVP68NMJ902M5A1)

--------------------------------------------------------------------------------------------------------------------------------------------

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Once _Advanced Analytics_ is enabled users need the _View advanced reporting_ permission to access this dashboard.

Why?[](https://support.bynder.com/hc/en-us/articles/28989709007890-Understanding-The-CX-Omnichannel-Dashboard#h_01K3G5RPV0B8T6ED8NCAY17SJB)

--------------------------------------------------------------------------------------------------------------------------------------------

It's important for users to be able to see their return on investment in their Bynder CX Omnichannel offerings. Using the CX Omnichannel dashboard users can assess and optimize the performance of the digital assets across external websites. This data shows users several metrics, such as how many transformations have been used, what assets are most used, and where the assets are appearing on the web.

How?[](https://support.bynder.com/hc/en-us/articles/25834920685714-Understanding-The-AI-Search-Dashboard-In-Advanced-Analytics#h_01JR30HXC0AYRX2193DHSS1G99)
[](https://support.bynder.com/hc/en-us/articles/28989709007890-Understanding-The-CX-Omnichannel-Dashboard#h_01JR30HXC0AYRX2193DHSS1G99)

The CX Omnichannel dashboard is incredibly granular. Users can decide on several levels how they wish to see the data.

**Time Frame:** Time Frame is delineated into three sections, the date-filter selector section, the numerical amount wanted, and the time-filter selector section.

|     |     |
| --- | --- |
| **Date Filter Options:** | **Time Filter Options:** |
| *   is in the last | *   seconds |
| *   is on the day | *   minutes |
| *   is in range | *   hours |
| *   is before | *   days |
| *   is on or after | *   weeks |
| *   is in the year | *   months |
| *   is in the month | *   quarters |
| *   is this | *   years |
| *   is next | *   complete seconds |
| *   is previous | *   complete minutes |
| *   is | *   complete hours |
| *   is null | *   complete days |
| *   is anytime | *   complete weeks |
| *   is not null | *   complete quarters |
| *   matches a user attribute | *   complete years |
| *   matches(advanced) |     |

For example, if you wanted to select data that _is in the last_ | _2_ | _quarters_ that is possible.

![Screenshot 2025-04-05 at 15.21.12.png](https://support.bynder.com/hc/article_attachments/28989724656146)

**Date Granularity:** Decide how you want the data presented between _Daily, Weekly, Monthly, Quarterly_ and _Yearly._

![Screenshot 2025-04-05 at 15.23.20.png](https://support.bynder.com/hc/article_attachments/28989724656658)

_Optional data refinement options are:_

**Asset:** To search a specific asset.

**Asset Type:** To search for a specific asset type.

**Collection:** To search for a specific collection.

**Serving Types:** To refine the search for _Adaptive Video, DAT, Public Links,_ or _Predictable URLs._

**Website:** To search from a list of websites populated by the usage of assets across the web.   
![](https://support.bynder.com/hc/article_attachments/28989709002130)  
Once you have decided what data you would like to see in the dashboard you can see the data for the following:

Directly below are the individual graphs for each available metric.

**Servings by Type over Time:** Shows users what and when assets are being used.

**Servings by Transformation over Time** and **Activity by Transformation:** Shows the users what transformations are being used, and how often.

### Download CX Omnichannel Dashboard Data

Downloading the dashboard data is a snap! Simply click the ![Screenshot 2025-04-05 at 15.49.09.png](https://support.bynder.com/hc/article_attachments/28990193484562)button. Once the menu opens click **Download.**

**![Screenshot 2025-04-05 at 15.48.10.png](https://support.bynder.com/hc/article_attachments/28990163139730)**

This will open a pop-up that allows you to configure the formatting of your downloaded data. You can view the data by pressing _Open in Browser_ or you can download by pressing _Download._

![](https://support.bynder.com/hc/article_attachments/28990193491730)

Bynder will generate your report and download the document to your device.
