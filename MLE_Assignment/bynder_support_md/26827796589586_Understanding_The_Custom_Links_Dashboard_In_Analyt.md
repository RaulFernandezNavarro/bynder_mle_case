# Understanding The Custom Links Dashboard In Analytics – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/26827796589586

Summary[](https://support.bynder.com/hc/en-us/articles/26827796589586#h_01JR303858N63CP7Z9R80PGZ9E)

----------------------------------------------------------------------------------------------------

Bynder offers extensive analytics options for our users. One such offering is the _Custom Links Dashboard_. This dashboard is configured to assist _Analytics_ users to get in-depth information on their users custom link data. This information can be provisioned with exceptional granularity as with all dashboards to allow maximum usefulness and customization of the data. Users can get data by search type, user group, and individual users to identify top performers and areas for improvement—helping you understand your users usage of custom links.

Who?[](https://support.bynder.com/hc/en-us/articles/26827796589586#h_01JR309BKFF5JJMV0WZFH0S3EZ)

-------------------------------------------------------------------------------------------------

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Once _Analytics_ is enabled users have to access this dashboard.

Why?[](https://support.bynder.com/hc/en-us/articles/26827796589586#h_01JR30B5FCN3E9T4GARSV2FAS8)

-------------------------------------------------------------------------------------------------

It's important for Bynder Admins to be able to see their return on investment in their Bynder enabled custom links. Using the _Custom Links Dashboard_ Bynder Admins can assess the usage of the custom links enabled in their portals. As an admin, user may want to understand which audiences are engaging with which content, ensuring the content is reaching the intended audiences, this dashboard will assist with this data procurement. 

How?[](https://support.bynder.com/hc/en-us/articles/26827796589586#h_01JR30HXC0AYRX2193DHSS1G99)

-------------------------------------------------------------------------------------------------

The Custom Links dashboard is delightfully granular. Users can decide on several levels how they wish to see the data.

_Note: Each analytics package offers different levels of offerings regarding the custom links dashboard, this will be noted below._

**Time Frame:** Time Frame is delineated into three sections, the date-filter selector section, the numerical amount wanted, and the time-filter selector section.

|     |     |
| --- | --- |
| **Date Filter Options:** | **Time Filter Options:** |
| *   is in the last | *   seconds |
| *   is on the day | *   minutes |
| *   is in range | *   hours |
| *   is before | *   days |
| *   is on or after | *   weeks |
| *   is in the year | *   months |
| *   is in the month | *   quarters |
| *   is this | *   years |
| *   is next | *   complete seconds |
| *   is previous | *   complete minutes |
| *   is | *   complete hours |
| *   is null | *   complete days |
| *   is anytime | *   complete weeks |
| *   is not null | *   complete quarters |
| *   matches a user attribute | *   complete years |
| *   matches(advanced) |     |

For example, if you wanted to select data that _is in the last_ | _2_ | _quarters_ that is possible.

![Screenshot 2025-04-05 at 15.21.12.png](https://support.bynder.com/hc/article_attachments/26827796577042)

**Date Granularity:** Decide how you want the data presented between _Daily, Weekly, Monthly, Quarterly_ and _Yearly._

![Screenshot 2025-04-05 at 15.23.20.png](https://support.bynder.com/hc/article_attachments/26827796577810)

**URL:** Decide what custom link you would like to pull the data for. Users can select _any value_ or a specific url to search.

**Destination:** Find custom links that go to specific locations within your DAM.

**QR Code:** Find custom links that have QR codes.

**Login Status:** Find where your custom links are being viewed from, the options are _Internal, External,_ or _Unknown._ 

**Password Protected:** See if the custom link is password protected or not.

![Screenshot 2025-04-05 at 15.25.23.png](https://support.bynder.com/hc/article_attachments/26827800346898)

Once you have refined what you would like to see in the dashboard you can see the data for the following:

![Screenshot 2025-04-05 at 15.36.33.png](https://support.bynder.com/hc/article_attachments/26827796580242)

**Clicks:** Total number of clicks. _(Available to all analytics users)_

**External Clicks:** Total number of clicks per external user. (_Available to Advanced Analytics users)_

**Internal Clicks:** Total number of clickers per internal user. (_Available to Advanced Analytics users)_

**Unique Links Clicks:** Total number of unique links clicked. (_Available to Advanced Analytics users)_

**Clicks per Link:** Total number of clicks per link. (_Available to Standard and Advanced Analytics users)_

**Links Created:** Number of links created in the portal. (_Available to Advanced Analytics users)_

Next users will see the _Clicks by Destination over Time_ chart. This is a breakdown of the total number of clicks by destination through time.

![Screenshot 2025-04-05 at 15.41.18.png](https://support.bynder.com/hc/article_attachments/26827800347282)

Then the following graphs will show the _Clicks by Destination and Login Status, Clicks by QR, and Clicks by Password._

![Screenshot 2025-04-05 at 14.50.47.png](https://support.bynder.com/hc/article_attachments/26827796582674)

The next section is _Clicks by Destination URL._ This graph shows the total number of clicks per destination URL. (The private information has been redacted for security purposes.)

![Screenshot 2025-05-20 at 10.41.35.png](https://support.bynder.com/hc/article_attachments/26829999831314)

The following section, _Clicks by Country,_ will be an interactive/clickable world map showing the total number of clicks by country. Simply scroll over an area to see the number of clicks for that country.

![Screenshot 2025-05-20 at 10.45.29.png](https://support.bynder.com/hc/article_attachments/26829999833106)

Following this is a section for _Clicks by Device over Time_ and _Clicks by Device_, which shows the total number of clicks per device type. 

![Screenshot 2025-05-20 at 10.52.12.png](https://support.bynder.com/hc/article_attachments/26829999837074)

The last section of this dashboard is the _Custom Links Created By Destination over Time_, which will show the total number of links made over time.

### Download Custom Links Dashboard Data

Downloading the dashboard data is a snap! Simply click the ![Screenshot 2025-04-05 at 15.49.09.png](https://support.bynder.com/hc/article_attachments/26827796584978) button. Once the menu opens click **Download.**

**![Screenshot 2025-04-05 at 15.48.10.png](https://support.bynder.com/hc/article_attachments/26827800352530)**

This will open a pop-up that allows you to configure the formatting of your downloaded data. You can view the data by pressing _Open in Browser_ or you can download by pressing _Download._

![Screenshot 2025-04-05 at 15.49.50.png](https://support.bynder.com/hc/article_attachments/26827800354578)

Bynder will generate your report and download the document to your device.
