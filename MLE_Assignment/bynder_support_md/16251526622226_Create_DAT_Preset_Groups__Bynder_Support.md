# Create DAT Preset Groups – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/16251526622226-Create-DAT-Preset-Groups

Within Dynamic Asset Transformations, you can create and assign [DAT presets](https://support.bynder.com/hc/en-us/articles/5071969529874)
 to a Group. Grouping DAT Presets allows you to organize the preset into categories based on your needs, roles, or asset types. For example, you might want to have a Social Media preset group that includes your Facebook, Instagram, TikTok, and YouTube DAT presets.

### Who can create a DAT Preset Group

Users with the following permissions can create DAT Preset Groups:

*   Manage Presets

Create a DAT Preset Group[](https://support.bynder.com/hc/en-us/articles/16251526622226-Create-DAT-Preset-Groups#h_01HKN6JBYDNBARC3A2X8JNCJD9)

-----------------------------------------------------------------------------------------------------------------------------------------------

1.  Navigate to Asset Bank.
2.  Select **Advanced** **Settings ![webdam_settings.png](https://support.bynder.com/hc/article_attachments/16326868357650)** \> **Portal** **Settings** > **DAT** **Presets.**![](https://pb-files.s3.amazonaws.com/production/attachments/03936f2a010d5b9c63bab7b88d6b4fefa361b01db87bf1a2c7757fe6b5a84f59/image.png)
3.  In the right-hand corner select **New preset group.**
4.  Name the **Preset** group.
    *   When naming your preset please include **\_** between each word and not a space.
5.  Click **Save**.![Screenshot 2024-01-08 at 2.22.04 PM.png](https://support.bynder.com/hc/article_attachments/16252131878034)
6.  Either create a [new DAT preset](https://support.bynder.com/hc/en-us/articles/5071969529874)
     or select an existing one and select **move to preset group.** ![Screenshot 2024-01-08 at 2.23.48 PM.png](https://support.bynder.com/hc/article_attachments/16252123235346)

View a DAT Preset Group[](https://support.bynder.com/hc/en-us/articles/16251526622226-Create-DAT-Preset-Groups#01HKN7DYEQNA2KET6HMXJCZBW7)

-------------------------------------------------------------------------------------------------------------------------------------------

1.  Navigate to Asset Bank.
2.  [Search](https://support.bynder.com/hc/en-us/articles/360013870360)
     and [filter](https://support.bynder.com/hc/en-us/articles/360013870120)
     for the asset. 
3.  In Asset detail, navigate to your transformations. 
4.  Here, you will see the transformations grouped together within their related Preset Group. 
5.  Click ![Screenshot 2024-01-11 at 9.13.03 PM.png](https://support.bynder.com/hc/article_attachments/16326868358674)  next to the DAT Preset Group to see the individual Preset.  ![](https://pb-files.s3.amazonaws.com/production/attachments/6de31ef57ebb278faaceff0d136aecaaa98b0c98e8b3d742a94d32c683a0634d/image.png)

Manage Presets in Group[](https://support.bynder.com/hc/en-us/articles/16251526622226-Create-DAT-Preset-Groups#h_01HKN8RJYQ8EBB8VETT9SCEPN4)

---------------------------------------------------------------------------------------------------------------------------------------------

1.  Navigate to Asset Bank.
2.  Select **Advanced** **Settings**\> **Portal** **Settings** > **DAT** **Presets.** 
3.  Navigate to the DAT Preset you want to add to the DAT Presets Group.
4.  Click![Screenshot 2024-01-08 at 2.29.48 PM.png](https://support.bynder.com/hc/article_attachments/16252131889170) next to the preset.
5.  Here, you can **move to a preset group,** or if it exists in a group already, you can select **remove from the group.** ![Screenshot 2024-01-08 at 2.30.06 PM.png](https://support.bynder.com/hc/article_attachments/16252123241618)

Rename Preset Group[](https://support.bynder.com/hc/en-us/articles/16251526622226-Create-DAT-Preset-Groups#01HKN7BH1TH85152W10JFGPY34)

---------------------------------------------------------------------------------------------------------------------------------------

1.  Navigate to Asset Bank.
2.  Select **Advanced** **Settings**\> **Portal** **Settings** > **DAT** **Presets in the top right corner**.
3.  Select the **Preset** **Group** you would like to edit and click ![Three dots.png](https://support.bynder.com/hc/article_attachments/16252131893394).
4.  You can **rename** the group
5.  Click **Save Changes**.  

Delete Preset Group[](https://support.bynder.com/hc/en-us/articles/16251526622226-Create-DAT-Preset-Groups#h_01HKN8NF3XAXCR82Z8JSQYY5Z9)

-----------------------------------------------------------------------------------------------------------------------------------------

1.  Navigate to Asset Bank
2.  Select **Advanced** **Settings**\> **Portal** **Settings** > **DAT** **Presets.** 
3.  Select **remove** **group** or **remove** **group** and **presets**.![Screenshot 2024-01-08 at 2.17.12 PM.png](https://support.bynder.com/hc/article_attachments/16252123248786)
4.  If you select **remove group,** the presets will remain in the preset list. If you select **remove group and presets,** this will remove both the group and presets.

Additional Information[](https://support.bynder.com/hc/en-us/articles/16251526622226-Create-DAT-Preset-Groups#h_01HKXQDTQYFKAADBXRSXT5BFFJ)

--------------------------------------------------------------------------------------------------------------------------------------------

A DAT preset can only belong to one Preset Group. You can duplicate a preset to add it to more than one group.
