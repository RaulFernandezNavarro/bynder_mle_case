# Duplicate Finder during Upload – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/13812491746194-Find-duplicates-during-upload

The Duplicate Finder during upload gives you complete control over preventing and managing duplicate assets upon upload to the portal. You can identify and consolidate duplicate files during the upload stage to ensure the assets you access are unique and up-to-date. All [Bynder-supported file types](https://support.bynder.com/hc/en-us/articles/360013936359)
 (image, document, and video types) are scanned for duplicates. 

The Duplicate Finder at Upload is available in the [Main Uploader](https://support.bynder.com/hc/en-us/articles/360013870260)
 and [Waiting Room](https://support.bynder.com/hc/en-us/articles/13811974882962)
. It is not currently available when uploading assets via the [Mass Uploader](https://support.bynder.com/hc/en-us/articles/360013870140)
 or [Upload API.](https://developer-docs.bynder.com/sdks)

Learn more about checking for [duplicates in the Waiting Room](https://support.bynder.com/hc/en-us/articles/13811974882962)
 and checking for [duplicate images](https://support.bynder.com/hc/en-us/articles/360013870020)
 in the Asset Bank.

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

**Will an asset be marked duplicate even with a different file name?**

An identical asset will be marked as such, even if the file names differ. 

**What if the asset has different metadata?**

Metadata embedded into the file itself is also considered when checking for duplicates. If there are differences, the images will not be picked up. For example, two identical photos, one with embedded GPS coordinates and the other not having them, will not be considered duplicates.

**Will users' permission to see certain assets be respected?**

If users have restricted visibility to assets, they will see the '**Show duplicates'** button but may not see the identified duplicate assets in the Asset Overview.

**Will all asset versions be scanned for duplicates?**

No, only active asset versions are scanned for duplicates.
