# Add Collections to Guides and Landing Pages – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/7382396334482

Collections can be added to Pages within [Guides](https://support.bynder.com/hc/en-us/articles/360013932239)
 and [Landing Pages](https://support.bynder.com/hc/en-us/articles/21908795341714)
 to quickly and easily share related assets with your users. To [display collections](https://support.bynder.com/hc/en-us/articles/22046924964242)
, please [publish](https://support.bynder.com/hc/en-us/articles/15954306339474)
 them online or [create the embed code](https://support.bynder.com/hc/en-us/articles/15954816230930)
. 

### Note

Users will receive an **Access** **Denied** message if a collection is [not made public](https://support.bynder.com/hc/en-us/articles/15954306339474)
 or shared with a specific user, user group, or permission profile.

**Who can create and customize content in guide and landing pages?**

How to Add Collections to Landing Pages or Guides[](https://support.bynder.com/hc/en-us/articles/7382396334482#h_01JD3008T787NQ4D5VC0T01WZV)

---------------------------------------------------------------------------------------------------------------------------------------------

1.  Navigate to your Landing Page or Guide.
2.  Select **edit.**
3.  Click + and select **![](https://support.bynder.com/hc/article_attachments/22737922010514)**to add a collection. _Collections and assets must not be watermarked to allow for downloads._
4.  Select the Collection > **Add** **Collection**. 
5.  You will first see an overview of collections Published online.
6.  Click the **dropdown** to view the **Embedded** and **Published** online Collections. ![](https://support.bynder.com/hc/article_attachments/22045624530066)
7.  Browse or search for the collection you want to add. 
    *   If you need to access your Collections, select the **Collection area**, where you will be directed to your Collections. 
8.  Select the Collection and click **Add** **Collection**.
    

_Note: Collections embedded in the guides or landing pages will only show up to 250 assets. Users must open the collection to see the entirety of the assets available._

Customize Collection Widget[](https://support.bynder.com/hc/en-us/articles/7382396334482#h_01JCNYW4PEZE4HQG335AN618PF)

-----------------------------------------------------------------------------------------------------------------------

Once you've selected the collection, you can change the layout drive, assets shown, and collection thumbnail, all within the **Widget** tab.

*   *   **Layout grid:** Masonry or Grid
    *   **Assets shown:** Number of assets shown
    *   **Font** **Color** and **size** (of Collection title and asset count)
    *   **Asset:** Whether or not you want the Collection title or type of files to appear in the details
    *   **Download:** If you want users to be able to download **assets** or **Collections**.
    *   **Redirection**: If you want the user to be redirected to the **asset** or the **Collection** toggle on.
        *   If **redirect to asset** is enabled, the end-user can visit an asset inside the asset bank directly from the guide by clicking on it.
        *   If **redirect** **to collection** is enabled, the end user can visit the collection directly from the guide by clicking on the **Collection** **title.**
    *   If both settings are **disabled**, the collection title and assets inside will not be clickable.
    *   **Default** **option**: Asset and collection redirection are enabled.

**What happens when a user clicks on a Collection included in a Guide or Landing Page?**

When a Guide or Landing Page includes a shared Collection, you can easily browse and download its assets, provided the permissions allow it.

If you are a Bynder user, please log in to maximize your access to individual asset downloads.
