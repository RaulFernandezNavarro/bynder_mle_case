# How To Restore Deleted Assets, Metaproperties, And Metaproperty Options – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013875320-How-To-Restore-Deleted-Assets-Metaproperties-And-Metaproperty-Options

Summary[](https://support.bynder.com/hc/en-us/articles/360013875320-How-To-Restore-Deleted-Assets-Metaproperties-And-Metaproperty-Options#h_01JXSRNN7HFST3B3RY9G621R9Z)

------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Bynder understands that accidents happen. If you accidentally delete assets, metaproperties, or metaproperty options, you can restore them as long as you do so before 30 days. There are some things in Bynder that _cannot_ be restored, fortunately, assets, metaproperties, and metaproperty options can be restored after accidental deletion. 

Who?[](https://support.bynder.com/hc/en-us/articles/360013875320-How-To-Restore-Deleted-Assets-Metaproperties-And-Metaproperty-Options#h_01JXSRR2ZA1XZBH31Y95F8ME1G)

---------------------------------------------------------------------------------------------------------------------------------------------------------------------

Users with the _Restore removed assets_ permission can restore removed assets. Users with the _Restore removed metaproperties_ can restore removed metaproperties, and users with the _Restore removed metaproperty options_ permission can restore removed metaproperty options.

How?[](https://support.bynder.com/hc/en-us/articles/360013875320-How-To-Restore-Deleted-Assets-Metaproperties-And-Metaproperty-Options#h_01JXSRWCDGRS2XKMW95HCCHD8M)

---------------------------------------------------------------------------------------------------------------------------------------------------------------------

If an account has multiple portals, you must follow the below steps for _every_ portal separately to see deleted files.

1.  **Navigate** to your Portal. 
2.  Click **Settings** > **Advanced** **Settings** > **Trash.**![Screenshot 2024-02-09 at 1.08.47 PM.png](https://support.bynder.com/hc/article_attachments/16934717289106)
3.  In the lefthand sidebar, select from the options **Assets**, **Metaproperties,** or **Metaproperty options. ![Screenshot 2024-02-09 at 1.24.54 PM.png](https://support.bynder.com/hc/article_attachments/16934675793554)** 
4.  Filter using **Removed by** and type in the user name to see which files that individual deleted. 
5.  Select the item you would like to restore and click the ![restore_trash.png](https://support.bynder.com/hc/article_attachments/10640557731346) button.  
    
    ![Screenshot 2024-02-09 at 1.28.25 PM.png](https://support.bynder.com/hc/article_attachments/16934675800338)
    
    The files are restored to the Asset Bank or the Waiting Room with all the linked Metadata, permissions, additional files, derivatives, and DAT (if associated with the asset).
    *   Assets will reappear in search results, and if they were included in a Collection, they will be restored to the appropriate Collection.
    *   Metaproperties and metaproperty options return to the state they were in before they were deleted.
    *   All related permission settings, language labels, and metaproperty images are restored.
        *   Restoring the deleted metaproperty options of other deleted metaproperty options does not bring back the main metaproperties.
    *   DAT links will automatically repopulate.

*   The triggers and metaproperty dependencies set up for metaproperty options will not be restored.
*   Files deleted from Asset Workflow or Asset Workflow metaproperties will not be restored. Get in touch with your Customer Success Contact if you need these restored.
