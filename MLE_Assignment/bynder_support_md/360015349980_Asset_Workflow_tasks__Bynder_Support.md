# Asset Workflow tasks – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360015349980-Asset-Workflow-tasks

When creating a Asset Workflow preset you may have some tasks that are not worth creating an individual stage for. This is where Asset Workflow tasks come in. They allow you to create a to-do list for a specific stage. You can assign tasks to specific users and make them required so that the job can only be submitted to the next stage when all tasks are completed.

How to Create a Task[](https://support.bynder.com/hc/en-us/articles/360015349980-Asset-Workflow-tasks#h_01HKYE65BABPR5QMCW492Q5CB2)

------------------------------------------------------------------------------------------------------------------------------------

1.  Go to ![settings_menu.png](https://support.bynder.com/hc/article_attachments/10108181930386)**Settings > Asset Workflow > Preset Management** and open the preset you want to add tasks to.
    
2.  Scroll to the stage where you want to add tasks to.
    
3.  Click **Add new** at the bottom of the stage in the **Tasks** section and enter the task you want to add.
    
4.  (Optional) Click **Unassigned** to enable extra options:
    
    *   **Assign this task to**
        
        By default, a task is not assigned to a specific user and can be completed by any stage responsible. Use the dropdown to assign a user or user group as task responsible. The responsible user(s) will receive an email notification when they are responsible for completing a task.
        
    *   **Mark task as important**
        
        Tick **Mark task as important** when this task requires more attention than other tasks. The task will be indicated by an ![exclamation.png](https://support.bynder.com/hc/article_attachments/10108178771090) icon.
        
    *   **Mark task as required**
        
        Tick **Mark task as required** when you want to make it obligatory for users to complete the task first before the job can be submitted to the next stage. Required tasks are indicated by an **\***.
        
    *   **Choose responsible when creating the job**
        
        Choose this option if you do not want to assign a designated responsible user to this task. The user creating the job will instead be able to assign a responsible user to this task.
        
    *   **Restrict choice to selected group**
        
        If you ticked the previous option, the job creator can assign any user as task responsible. Tick the **Restrict choice to selected group** option if you want the job creator to only be able to select a task responsible from a specific user group. Click the **Select a group** dropdown, search for the group, and click the name of the group to add it. 
        
    *   **Set the due date**
        
        If the task needs to be completed on a specific date, you can use the date picker to set up a due date.
        
    
5.  Click **Add this task** to add the task to your Asset Workflow preset.
    
6.  Click the **Save** button at the bottom of the page to save your preset.
    

### Note

Fuzzy search is not supported when working with special characters for specific languages.  

How to Edit a Task[](https://support.bynder.com/hc/en-us/articles/360015349980-Asset-Workflow-tasks#h_01HKYE65BAWC3E6P7J750A4MXD)

----------------------------------------------------------------------------------------------------------------------------------

1.  Go to ![settings_menu.png](https://support.bynder.com/hc/article_attachments/10108181930386)**Settings > Asset Workflow > Preset Management** and open the preset where you can find the task you want to edit.
    
2.  Scroll to the stage with the task that has to be modified.
    
3.  Hover over the task you want to edit and click the ![edit-metadata_pencil.png](https://support.bynder.com/hc/article_attachments/10108178865170) icon.
    
4.  Make the necessary changes and click **Save this task** to save the changes.
    
5.  Click the **Save** button at the bottom of the page to save your modified preset.
    

How to Remove a Task from a preset[](https://support.bynder.com/hc/en-us/articles/360015349980-Asset-Workflow-tasks#h_01HKYE65BAX0N4GRDQG1W3GDVN)

--------------------------------------------------------------------------------------------------------------------------------------------------

1.  Go to ![settings_menu.png](https://support.bynder.com/hc/article_attachments/10108181930386)**Settings > Asset Workflow > Preset Management** and open the preset where you can find the task you want to remove.
    
2.  Scroll to the stage with the task that has to be deleted.
    
3.  Hover over the task you want to edit and click the ![203285644-trash_-_blue__1_.png](https://support.bynder.com/hc/article_attachments/10108182137106) icon.
    
4.  Click **Save this task** to save the changes and click the **Save** button at the bottom of the page to save your modified preset.
