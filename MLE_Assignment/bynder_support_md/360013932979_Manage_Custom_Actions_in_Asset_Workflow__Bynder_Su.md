# Manage Custom Actions in Asset Workflow – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013932979-Manage-Custom-Actions-in-Asset-Workflow

With custom actions, you can to skip stages in a workflow job. For example, you can go from stage 1 to stage 3 or from stage 5 to stage 2.

![result-custom-action.png](https://support.bynder.com/hc/article_attachments/10108144599570)

1.  Go to **Settings > Workflow > Preset management**.
    
2.  Click the **Job** hyperlink and select the preset to which you want to add a custom action.
    
3.  Go to the stage from which you want to skip to a previous or a next stage and select **Use custom actions**.
    
4.  Select the stage the custom action will direct to, define the label that describes the action and pick the color of the button.
    
    ![custom-action.png](https://support.bynder.com/hc/article_attachments/10108175007506)
    
5.  When you create one custom action, the normal **Send to the next stage** and **Send to the previous stage** buttons will disappear for that stage. Make sure you create the going forward and the going back buttons in Custom actions.
    
    ![custom-action-two.png](https://support.bynder.com/hc/article_attachments/10108144684690)
