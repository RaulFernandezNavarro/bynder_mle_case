# Create Variables in Print Brand Templates – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013873160-Create-Variables-in-Print-Brand-Templates

You have the option to include variables in your Print Brand Templates. This allows users to choose a variable from a drop-down menu while working with their template.

![Screen Shot 2017-06-29 at 14.01.23.png](https://support.bynder.com/hc/article_attachments/10089931678738)

**Who can manage Variables in Print Brand Templates?**

Users with the following right can create, manage and remove variables:

*   **Create template account variables**
    
*   **Delete template account variables**
*   **View template account variables**

1.  Go to **Settings > Brand Templates> Variable Management**.
    
2.  Specify the variable name.
    
3.  Decide on the variable type:
    
    *   **Single**: provide a single value or multiple values separated by a coma.
        
    *   **Collection**: add one or more options. Drag and drop the options to change their hierarchy. Also, creating a collection variable allows for creating values with a coma as a part of the value.
        
    

Now, you can select this variable name in the settings of your POD template.
