# AI Search For Universal Compact View (UCV) – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/28988145584530-AI-Search-For-Universal-Compact-View-UCV

Summary[](https://support.bynder.com/hc/en-us/articles/28988145584530-AI-Search-For-Universal-Compact-View-UCV#h_01K3G3BKFWZHJ8C94DXT8J5NWH)

---------------------------------------------------------------------------------------------------------------------------------------------

Bynder’s Universal Compact View is a configurable, pre-built UI component providing a window into your DAM for searching and selecting assets from within another system. UCV can be configured to include Bynder's _Natural Language Search_ and _Similarity Search_ to further optimize asset management. Used in many of our existing integrations, it is easily implemented, saving time and money, and accelerating the process of developing integrations into other systems. 

Who?[](https://support.bynder.com/hc/en-us/articles/28988145584530-AI-Search-For-Universal-Compact-View-UCV#h_01K3G3FD3QF6T51M0V9VBQKXDF)

------------------------------------------------------------------------------------------------------------------------------------------

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Why?[](https://support.bynder.com/hc/en-us/articles/28988145584530-AI-Search-For-Universal-Compact-View-UCV#h_01K3G3MSJQNSS0ENJ980SZC0HJ)

------------------------------------------------------------------------------------------------------------------------------------------

Users may need to find assets based on less tangible qualities than what is saved as metadata. They may need to search based on a phrase, with Bynder's _Natural Language Search (NLS)_ built into the UCV users can leverage Bynder's AI technology to find assets with ease and efficiency. Furthermore, if users need to find an image that is similar to one they are looking at users can employ Bynder's _Similarity Search_ feature, allowing users to get a list of similar image assets directly from the UCV.

How?[](https://support.bynder.com/hc/en-us/articles/28988145584530-AI-Search-For-Universal-Compact-View-UCV#h_01K3G4280MNK69WDQC85F8ZAF1)

------------------------------------------------------------------------------------------------------------------------------------------

### Natural Language Search For UCV

1.  From the UCV, users type their query using natural language such as "an excited group of friends" into the main search bar. 
    
2.  Toggle the NLS search to the _on_ position. ![UCV.png](https://support.bynder.com/hc/article_attachments/28988621074578)
    
3.  Send the query. 
    
4.  The system leverages AI to understand the query and returns visually relevant images.
    

### Similarity Search For UCV

1.  Open the UCV, select a source image, and select "Find similar assets" from the footer.
    
2.  The UCV returns assets that share visual characteristics with the chosen image, allowing the user to find the assets they need quickly.
