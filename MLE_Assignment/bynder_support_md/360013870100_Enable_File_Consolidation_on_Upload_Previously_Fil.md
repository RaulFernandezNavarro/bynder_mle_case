# Enable File Consolidation on Upload (Previously File Merge) – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013870100-Enable-File-Consolidation-on-Upload-Previously-File-Merge

File Consolidation allows you to [upload multiple files](https://support.bynder.com/hc/en-us/articles/24475776405394)
, such as PDFs and JPEGs, as a single asset rather than separate files. The system designates one file as the main asset while keeping the others grouped together as additional files. This helps streamline asset management by ensuring related files stay connected under one entry.

By grouping similar files, File Consolidation keeps the Asset Bank organized and reduces duplicate uploads, saving time and effort. It also improves searchability since all related files are stored in one place. This feature simplifies workflows, especially when managing different formats or translations, and supports approval processes by allowing files to be sent for review when needed.

**Who can manage and configure the file consolidation settings**

What Happens to Existing  Merged Files[](https://support.bynder.com/hc/en-us/articles/360013870100-Enable-File-Consolidation-on-Upload-Previously-File-Merge#h_01JKEYNFANAJG1Y2R9BPV2FEMR)

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

All previously merged files will **remain unchanged** in your Asset Bank. Activating File Consolidation will **disable** File Merge, and files will be consolidated according to your settings.

How to Enable This Feature [](https://support.bynder.com/hc/en-us/articles/360013870100-Enable-File-Consolidation-on-Upload-Previously-File-Merge#h_01JKEYXQW8EXQSCJQP5DAAPNQM)

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1.  Navigate to **Portal Settings > File Consolidation**.![](https://support.bynder.com/hc/article_attachments/24475683797778)
2.  Activate **File Consolidation** (this will **disable** File Merge if enabled).
3.  Set up **MIME type priority—**the main file will be the first-listed file type, while others will be **additional files**.

How to Configure MIME types[](https://support.bynder.com/hc/en-us/articles/360013870100-Enable-File-Consolidation-on-Upload-Previously-File-Merge#h_01JKEYNFANR512ARKN2KJA18HT)

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1.  Select [**Add MIME type**](https://support.bynder.com/hc/en-us/articles/360013870100-Enable-File-Consolidation-on-Upload-Previously-File-Merge#h_01JKEYNFAN508QGZT4SRQ47JCX)
     to add a new type > **Save**.
2.  Drag and drop MIME types to reorder.
3.  Use the three-dot menu next to the name of the **MIME** type
    *   You can either **edit** or delete a **MIME** type.  

Supported MIME Types[](https://support.bynder.com/hc/en-us/articles/360013870100-Enable-File-Consolidation-on-Upload-Previously-File-Merge#h_01JKEYNFAN508QGZT4SRQ47JCX)

**Can I use File Consolidation when uploading assets via the External Uploader?**

No, File Consolidation only applies to uploads via the [front-end uploader.](https://support.bynder.com/hc/en-us/articles/360013870260)
 Learn more [here](https://support.bynder.com/hc/en-us/articles/24475776405394)
.

**Can I change the main file after using File Consolidation?**

No, the main file is permanently assigned once consolidation occurs. Additional files remain supplementary and cannot be reassigned as the main file.

**What happens to existing merged files?**

Previously merged files will remain unchanged in the Asset Bank. However, enabling File Consolidation will automatically disable the File Merge feature.
