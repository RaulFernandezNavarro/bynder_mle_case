# Custom Job Overview in Asset Workflow – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360015363199-Custom-Job-Overview-in-Asset-Workflow

Create a Custom Job Overview to simplify job search and organization for Workflow users. This customized overview allows filtering by [**Workflow metaproperty**](https://support.bynder.com/hc/en-us/articles/360013871580)
 to easily locate open jobs based on specific criteria.

The job overview includes default filters in your [Asset Workflow](https://support.bynder.com/hc/en-us/articles/360013932859)
. These predefined filters are: campaign, responsible, involvement, reporter, text, and status.

### Note

Share a specific filtered view with another user in the Asset Workflow by applying filters in the custom job overview, then simply copy and share the URL

7.  Select **Save.**

**Can I switch back to the default job overview screen?**

Yes, to switch back to the default filter setup, remove all custom job filters by clicking the **x** next to each one.

1.  Remove all custom column filters by clicking the **x** next to each custom column.  
    **Note**This action is permanent and cannot be undone.
2.  Click **Save** to save the changes and switch to the default filter and column setup.
