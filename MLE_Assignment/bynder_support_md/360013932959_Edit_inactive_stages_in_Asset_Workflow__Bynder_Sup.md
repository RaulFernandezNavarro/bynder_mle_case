# Edit inactive stages in Asset Workflow – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013932959-Edit-inactive-stages-in-Asset-Workflow

You can select an option to edit inactive stages in workflows. These editing rights mean that users can remove attachments and assets, edit form fields, add and remove tasks, or edit or remove stages. Uploading assets to inactive stages is not possible.

The edit rights work in combination with the **Editable by** field in the workflow. See [Assign edit rights to stages.](https://support.bynder.com/hc/articles/360013871420#UUID-55ea0c7d-604d-df8a-6cbf-87d76f2407fe "Assign edit rights to stages")

The customer can now decide which user profiles can edit workflow stages.

How to enable this feature?[](https://support.bynder.com/hc/en-us/articles/360013932959-Edit-inactive-stages-in-Asset-Workflow#h_01HP2KAGA16KXW8BV5M2K923HR)

-------------------------------------------------------------------------------------------------------------------------------------------------------------

Contact your Customer Success Contact if you would like to have this feature enabled for your portal.

Try it[](https://support.bynder.com/hc/en-us/articles/360013932959-Edit-inactive-stages-in-Asset-Workflow#h_01HP2KAGA16W9BBK2NT5R9XC9C)

----------------------------------------------------------------------------------------------------------------------------------------

1.  Go to **Settings > Users & rights > Permission Management**.
    
2.  Select the profile which you want to be able to edit inactive stages.
    
3.  Enable the **Manage inactive stages** permission.
    
4.  Click **Save** to save the changes.
    

Results[](https://support.bynder.com/hc/en-us/articles/360013932959-Edit-inactive-stages-in-Asset-Workflow#h_01HP2KAGA1J6S519H3A9ZVH4ZP)

-----------------------------------------------------------------------------------------------------------------------------------------

The selected users can now edit inactive stages in the workflow. Depending on the option selected in the **Editable by** field, these can be all the users involved in the workflow, only the person responsible for a particular stage, or the person responsible for the stage and the person accountable for the whole workflow job.

Learn more[](https://support.bynder.com/hc/en-us/articles/360013932959-Edit-inactive-stages-in-Asset-Workflow#h_01HP2KAGA1759TG2ZARW9KMXHN)
