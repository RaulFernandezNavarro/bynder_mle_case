# Create Metaproperty Options – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/10341416276370

Metaproperty options belong to specific [metaproperties](https://support.bynder.com/hc/en-us/articles/6190351879186)
 and play a big role in your portal's taxonomy. They allow you to classify your assets better and help users find the assets they seek.

Metaproperties and their options are essential to your Bynder taxonomy and should be configured before you [upload](https://support.bynder.com/hc/en-us/articles/360013931479)
 your assets. You cannot add options for the metaproperties of linked **assets, text, and button types.** For example, if you have a **Country** metaproperty, you can add countries as options to this metaproperty. Once you have set up your metaproperty option, you can [change them.](https://support.bynder.com/hc/en-us/articles/19463374475154)

**Who can manage metaproperties and options?**

Users with the following rights can manage metaproperties:

*   [**Manage metaproperties**](https://support.bynder.com/hc/en-us/articles/18433246503442-Permissions-in-Asset-Bank#:~:text=Manage%20metaproperties%20%2D%20Allows%20users%20to%20manage%20metaproperties.)
    
*   [**Manage metaproperty options**](https://support.bynder.com/hc/en-us/articles/18433246503442-Permissions-in-Asset-Bank#:~:text=Manage%20metaproperty%20options%20%2D%20Allows%20users%20to%20manage%20metaproperty%20options.)
    

Create Metaproperty Options[](https://support.bynder.com/hc/en-us/articles/10341416276370#h_01H966EPE24FA5H0WQXRYQC2HA)

------------------------------------------------------------------------------------------------------------------------

Once you've successfully [set up your metaproperties](https://support.bynder.com/hc/en-us/articles/6190351879186)
, you can add options.

1.  Navigate to ![settings_menu.png](https://support.bynder.com/hc/article_attachments/10433893989266) **Settings** > **Taxonomy** > **Metaproperties** **management**\> **Metaproperties** **and** **options** in the left sidebar.
    
2.  Navigate or search for the metaproperty you want to add an option to, or if you are creating a new metaproperty, you can begin adding metaproperty options in the same window.
    
3.  Click **\+ Add Option**.
    
4.  Under **Options,** enter the option name and click **Add Option.**
    
    *   Note: The "unique name" cannot be changed; however, you can change the label if you decide to change the name of the selected metaproperty. See [here](https://support.bynder.com/hc/en-us/articles/22711704142866)
         for naming guidelines.
    
    **![Screenshot 2024-11-07 at 10.58.42 AM.png](https://support.bynder.com/hc/article_attachments/22485879210002)**
    
5.  Click **Save**.![Screenshot 2024-11-07 at 10.59.20 AM.png](https://support.bynder.com/hc/article_attachments/22485895462418)
    
6.  Once you've added all the options, you can customize your metaproperty options, such as adding [access rights](https://support.bynder.com/hc/en-us/articles/360013872840-Set-Permissions-on-a-Metaproperty-Option-Level)
    , [multiple languages](https://support.bynder.com/hc/en-us/articles/10341416276370#h_01JC3MH5ZYYAZN7WNY70XAM0FV)
    , [external references](https://support.bynder.com/hc/en-us/articles/10341416276370#01JC3N3M9J2BGWW40M4J00VZRH)
    , or [visuals](https://support.bynder.com/hc/en-us/articles/10341416276370#01JC3MWJR9P9QSE408F6D58HT4)
    .

### How to Add a Font Awesome Icon

1.  You can use Font Awesome icons, available [here](https://fontawesome.com/v4/icons/)
    , v4.7, and below.
    *   Icons from Font Awesome version 5 or above are not supported.
2.  Click ![](https://support.bynder.com/hc/article_attachments/11685696139794) to browse or search for icons.
3.  To use an icon from Font Awesome, click on it to get the full name and copy it.![fontawesome_example](https://support.bynder.com/hc/article_attachments/11685674790418)
4.  Click ![](https://support.bynder.com/hc/article_attachments/11685712322578) next to the **Font Awesome icon** and paste the name.
    *   The name of the icon can be up to 15 characters. 
5.  Click ![](https://support.bynder.com/hc/article_attachments/11685713467410) to save.

### How to Add a Thumbnail Image

1.  Click ![](https://support.bynder.com/hc/article_attachments/11685696139794) next to **Upload an image** to the metaproperty option icon page.
2.  Click + and select an image from your computer.
3.  The recommended size for images is 24 px with a 1:1 ratio.
4.  Click **Save.**

**Is it possible to add duplicate metaproperty options?**

No, you cannot use the same name for multiple metaproperty options that are listed under the same metaproperty. 

**Is there a limit to the number of metaproperty options I can add?**

No, there is no limit to the number of metaproperty options you can add. However, if you reach 10k+ options, it may impact the general speed of the portal.
