# Understanding And Using Metadata Presets – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013931319-Understanding-And-Using-Metadata-Presets

Summary[](https://support.bynder.com/hc/en-us/articles/360013931319-Understanding-And-Using-Metadata-Presets#h_01JWR6YA3JK5S0Z08WC4WTHQDD)

-------------------------------------------------------------------------------------------------------------------------------------------

You can create a metadata preset to easily add the correct metadata to your asset during the upload process. Metadata presets are pre-defined sets of metadata that can include descriptions, text fields, metaproperty options, advanced options, custom fields, and collections.  Metadata presets are only available during upload and not when batch editing assets, this means that they are not available in the Waiting Room either. Presets do not overwrite previous selections but add the data to the existing information.

Who?[](https://support.bynder.com/hc/en-us/articles/360013931319-Understanding-And-Using-Metadata-Presets#h_01JWR6YX9Y0GQT80QHTA8FY96F)

----------------------------------------------------------------------------------------------------------------------------------------

This feature/solution is enabled by a Bynder Admin.

Users with the _Share metadata presets_ permission can share and hide metadata presets.

Why?[](https://support.bynder.com/hc/en-us/articles/360013931319-Understanding-And-Using-Metadata-Presets#h_01JWR700JR27A2QRPDCZGTRP5A)

----------------------------------------------------------------------------------------------------------------------------------------

Presets can be applied to future uploads to add the correct metadata automatically. This helps to prevent mistakes and ensures that your assets are always properly tagged. You can share and restrict the visibility of these presets to certain users.

How?[](https://support.bynder.com/hc/en-us/articles/360013931319-Understanding-And-Using-Metadata-Presets#h_01HMCFHZHRFF0668B3X6Z9MKSG)

----------------------------------------------------------------------------------------------------------------------------------------

### Creating A Metadata Preset

1.  Go to your portal and click **Upload Media** to start the upload. 
    
2.  Use the left sidebar to tag your assets with the right metadata, metaproperty options, advanced options, custom fields, and collections.
    
3.  Go to the **Presets** section and click the ![add_metadata_preset.png](https://support.bynder.com/hc/article_attachments/10641646064018) **Create new preset** button.
    
4.  Enter a name for the preset in the **Enter new preset name** field.
    
5.  Click the **Create preset** button to create the new preset.
    
6.  Finish the upload, or cancel the upload, the preset is already saved.
    

_Note:_ It is imperative that users do these steps in this order for this task to be accomplished.

### Creating A Preset Based On An Existing One

1.  In the upload screen, select one of the existing presets. Your new preset will be based on this preset.
2.  Make the required updates/customizations in the lefthand sidebar.
3.  Click the ![add_metadata_preset.png](https://support.bynder.com/hc/article_attachments/10641646064018) **Create new preset** button.
4.  Enter a name for the new preset and click **Create preset**.

### Modifying A Metadata Preset

1.  Navigate to your **Portal**. 
    
2.  Select **Upload media** to start the upload.
    
3.  In the left sidebar, open the **Select a preset** dropdown and click the preset you want to modify.
    
4.  Use the left sidebar to make the necessary adjustments. Modify the metadata and select or deselect metaproperty options, advanced options, custom fields, and collections.
    
5.  Once all modifications have been made, open the **Preset** dropdown and click the ![update_metadata_preset.png](https://support.bynder.com/hc/article_attachments/10641633809426) **Update current preset** button to update the metadata preset.
    
6.  Finish the **upload**.
    

_Note: You can only delete presets that you've created. You cannot delete another user's preset._

### Applying A Metadata Preset

1.  Navigate to your **Portal**.
    
2.  Select **Upload media** to start the upload.
    
3.  In the left sidebar, **select a preset** dropdown and click the preset you want to apply to the selected assets.
    
4.  Finish the **upload.**
    

### Sharing Metadata Presets

1.  Navigate to your **Portal.**
    
2.  Select **Upload media** to start the upload. 
    
3.  Click the dropdown in the **Preset** section and click the preset you want to make available to other users.
    
4.  Click the ![settings_menu.png](https://support.bynder.com/hc/article_attachments/10641633902354) next to the dropdown and click **Preset visibility**.
    
5.  Make sure the visibility is set to **Specific users**. If not, click the **Change** dropdown and select **Specific Users.**
    
    **Specific Users:** _Metadata presets with this visibility can be used by the specified permission profiles, user groups, and users. The visibility screen recognizes these presets by their icons._
6.  Click the **Who has access** field and use the search bar to find the permission profile, user group, or user account with which you want to share the preset. Hover over the name and click the icon.
    
7.  (Optional) Repeat the previous step to share the preset with other permission profiles, user groups, or users.
    
8.  Click **Save** to share the metadata preset.
    

### Hiding Metadata Presets From Other Users

1.  Navigate to your **Portal.**
    
2.  Select and **Upload media** to start the upload.
    
3.  Click the dropdown in the **Preset** section and click the preset that you want to hide from other users.
    
4.  Click the ![settings_menu.png](https://support.bynder.com/hc/article_attachments/10641633902354) button next to the dropdown and click **Preset visibility**.
    
5.  Click the **Change** dropdown and select **Private**.
    
    **Private Users:** _Presets with this visibility can only be used and edited by the user who created the metadata preset. In the visibility screen, you can recognize these presets._
    
6.  Click **Save** to save the changes and hide the preset from other users.
    

_Note: Metadata presets **do not** overwrite any data selected previously. You can manually select some of the metaproperty options and then choose a preset. In this case, the data from the preset will be **added** **to** the previously selected data. Similarly, if you switch from one preset to another, any selection from the first preset will be kept, and extra information from the second preset will be added to the top._

_Presets **cannot** be reordered in a custom sequence._
