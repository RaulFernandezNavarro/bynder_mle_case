# Link Assets to Asset Workflow Jobs – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013871440-Link-Assets-to-Asset-Workflow-Jobs

If you need to track assets associated with specific workflow jobs, you can create metaproperties that will automatically connect these assets to the jobs. This allows you to easily filter and access specific workflow jobs directly from the asset details page.

How to enable this feature?[](https://support.bynder.com/hc/en-us/articles/360013871440-Link-Assets-to-Asset-Workflow-Jobs#h_01HMCKAY7D7DX8Y16SRNFN1DZC)

---------------------------------------------------------------------------------------------------------------------------------------------------------

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Link Assets to Asset Workflow Jobs 

1.  Contact your Customer Success Contact to create the new "Workflow Link" metaproperty that will link the assets to your Workflow Jobs. 
2.  Once the metaproperty has been created navigate to **Settings > Workflow > Preset Management**.
3.  Click the **Job** hyperlink.
    
4.  Click **\+ Add preset**.
    
5.  In the preset options, select **Add job to exported assets using the "Workflow link"** metaproperty.
    
6.  From the **Workflow link metaproperty** drop-down list, select the metaproperty you have created.
    
7.  Save your preset.
    

Create a Job Based on this Preset.[](https://support.bynder.com/hc/en-us/articles/360013871440-Link-Assets-to-Asset-Workflow-Jobs#h_01HVPTHGAKZ125KJGVV1RVRC4E)

----------------------------------------------------------------------------------------------------------------------------------------------------------------

When you create a job using a preset and upload assets to this job, the submitted assets in the Asset library will have a link to the original job. To see the link to the job, open the asset detail page.

1.  Navigate to Workflow.
2.  Click **Create new** and select the job preset you have created.
3.  Enter the name of the job and click **Add** to start the workflow job.
4.  Upload new assets.
5.  Export the assets to the **Waiting Room** or **Asset** **Bank**.
6.  In the Asset Detail view, select![Screen Shot 2016-10-12 at 10.37.08.png](https://support.bynder.com/hc/article_attachments/10108143381778) to view the link to the Workflow job.
