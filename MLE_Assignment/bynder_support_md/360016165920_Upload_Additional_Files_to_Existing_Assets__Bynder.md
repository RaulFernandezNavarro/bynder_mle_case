# Upload Additional Files to Existing Assets – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360016165920-Upload-Additional-Files-to-Existing-Assets

Additional files are supplementary files that can be included with assets. Examples include a release form or a price list. These files provide more detail or supporting information about an existing asset. 

By default, all users can view Additional Files; however, you can adjust permissions for certain users to [download additional files](https://support.bynder.com/hc/en-us/articles/360016165920-Upload-Additional-Files-to-Existing-Assets#h_01HETJES26RXG2CE40TSHZP979)
. 

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

### Who can upload additional files?

Users with the following permissions can upload additional files: 

*   Edit Assets
    

Upload Additional Files[](https://support.bynder.com/hc/en-us/articles/360016165920-Upload-Additional-Files-to-Existing-Assets#h_01HETJES26SWSQ0ZKT9KFN10CT)

-------------------------------------------------------------------------------------------------------------------------------------------------------------

1.  Navigate to the asset detail view of the asset to which you would like to add the file.
    
2.  Click the **Versions** tab on the right-hand side of the panel.
    
3.  Navigate to the **Additional** **Files** section and select ![add_metadata_preset.png](https://support.bynder.com/hc/article_attachments/10641688959250) **Click to add a file.**
    
4.  Select the file you want to upload as an additional file from your device.
    
    ### Note
    
    The maximum allowed file size of an additional file is 4GB.
    
5.  The additional file will now be available in the Asset Detail View.
    
6.  All assets with additional files are showing a paperclip-icon in the Asset Overview so users can easily identify assets that contain Additional files when browsing in the Asset Overview. Download Additional Files

**Who Can Download Additional Files**

1.  Navigate to the asset detail view of the asset with the additional file you want to download.
    
2.  In the **Info** tab, navigate to the **Additional** **Files** section.
    
3.  Click the additional file to start the download.
    
4.  (Optional) Alternatively, select the asset(s) from the Asset Bank and click the download icon. In the Download Preselect screen, check the box next to **Include all additional files** **in the download package.**
    
    *   If you want this option selected by default for all users, you can contact your Customer Success Contact.
        
    
    ![default_additional_files.png](https://support.bynder.com/hc/article_attachments/10641689024402)
    

Who Can Access Additional Files[](https://support.bynder.com/hc/en-us/articles/360016165920-Upload-Additional-Files-to-Existing-Assets#h_01HXF99RZXCRYVJ52QXN9W6NFG)

---------------------------------------------------------------------------------------------------------------------------------------------------------------------

If you have [**the download additional file**](https://support.bynder.com/hc/en-us/articles/360016165920-Upload-Additional-Files-to-Existing-Assets#h_01HETJES26RXG2CE40TSHZP979)
 feature enabled, then only users with the [Download asset additional](https://support.bynder.com/hc/en-us/articles/18433246503442-Permissions-in-Asset-Bank#:~:text=Download%20asset%20additionals%20%2D%20Allows%20users%20to%20download%20and%20view%20additional%20files.)
 permission can view **and** download additional files. If you don’t have the permission enabled users can neither see nor download the additional files

Delete Additional Files[](https://support.bynder.com/hc/en-us/articles/360016165920-Upload-Additional-Files-to-Existing-Assets#h_01HETJES26HS3H6MP1RXGS6EHB)

-------------------------------------------------------------------------------------------------------------------------------------------------------------

1.  Open the asset with the additional file you want to delete.
    
2.  Click the **Versions** tab on the right side of the panel and navigate to the **Additional** **Files** section.
    
3.  Click the![trash.png](https://support.bynder.com/hc/article_attachments/10641689038994) button next to the additional file you want to delete.
    
4.  This action cannot be undone.
    

Alternative Options to Additional Files[](https://support.bynder.com/hc/en-us/articles/360016165920-Upload-Additional-Files-to-Existing-Assets#h_01HETJES27YNTR7N54KEB7AHJ6)

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

If you would like to update an asset's original file without having to tag it from scratch, you might want to upload your updated file as a new version instead[.](https://support.bynder.com/hc/articles/360013931859#UUID-4a90b7e3-48b0-320d-b993-99a2e0f16777 "Upload a New Version")

If you want to add a new download option for users with specific customizations, such as crop or format, you might want to upload your modified file as a manual derivative instead[.](https://support.bynder.com/hc/articles/360013931159#UUID-92b4d184-3466-9e64-9fbb-ceb3ec49eaf3 "Manual derivatives")
  
You may want to use the [related assets](https://support.bynder.com/hc/en-us/articles/360013931679)
 functionality to identify assets that have been uploaded to the Asset Bank that have something in common.

Manual derivatives cannot be exposed via API and will not be available in any integration.
