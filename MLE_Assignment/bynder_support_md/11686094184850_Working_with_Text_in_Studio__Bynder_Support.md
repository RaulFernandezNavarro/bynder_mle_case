# Working with Text in Studio – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/11686094184850

Text is one of the elements that you can add to your designs within Studio. You will be able to control the various font and color options for your text in addition to what the content creators will be able to update when creating an image or video.

Add Text[](https://support.bynder.com/hc/en-us/articles/11686094184850#h_01HGB7C67V2ZQDG1M2JJ9WZHZ2)

-----------------------------------------------------------------------------------------------------

1.  Navigate to Studio and then select an existing design or create a new one. Learn more about getting started with [images](https://support.bynder.com/hc/en-us/articles/8572246980498)
     or [videos](https://support.bynder.com/hc/en-us/articles/9443675588882)
    .
2.  Add text: 
    *   **Images**: Click + new layer in the layer panel.
    *   **Videos**: Click + add asset in the timeline editor.
3.  Select **Text**, then enter the text copy.
4.  Learn more about the element options, such as how to lock, duplicate, rename or delete for [images](https://support.bynder.com/hc/en-us/articles/8572246980498)
     or [videos](https://support.bynder.com/hc/en-us/articles/9443675588882)
    .

Format Text[](https://support.bynder.com/hc/en-us/articles/11686094184850#h_01HGB7C67V0FSZHY1N3P3RY9YQ)

--------------------------------------------------------------------------------------------------------

1.  Click the Text element.
2.  **Edit text:** Double-click the text in the preview to edit the copy.
3.  **Text Element:**
    *   **Alignment**: Click ![](https://support.bynder.com/hc/article_attachments/11686031131922) to set the alignment within the preview or drag and drop the element to the desired location.
    *   **Resize**: Manually enter the width and height of the text element or drag and drop the corner to resize. Click ![](https://support.bynder.com/hc/article_attachments/11686018754194) to lock the aspect ratio.  
        ![](https://support.bynder.com/hc/article_attachments/11686010177810)
    *   **Location**: Enter the coordinates for the element or drag and drop it to the desired location.  
        ![](https://support.bynder.com/hc/article_attachments/11686052557842)
    *   **Rotate**: Enter the desired rotation or rotate the text box manually.  
        ![](https://support.bynder.com/hc/article_attachments/11686031631506)
    *   **Opacity**: Enter the desired opacity.  
        ![](https://support.bynder.com/hc/article_attachments/11686010332946)
4.  **Text**:
    *   **Text style** (optional): Select a text style from the list. Learn more about text styles [here](https://support.bynder.com/hc/en-us/articles/10294329447186)
        .
    *   **Font family:** Select the font family from the dropdown.
    *   **Font weight:** Select from weights that are part of the selected font family (if available).
    *   **Font size:** Set the size of the font.
    *   **Line height:** Control the vertical distance between lines of text in a paragraph.
    *   **Letter spacing:** Define the space between characters in a word, line, or paragraph.![](https://support.bynder.com/hc/article_attachments/16275727949714)![](https://support.bynder.com/hc/article_attachments/16275727953042)
    *   **Decoration: Underline or Strikethrough** Text
    *   **Script: Superscript or Subscript**
    *   **Case: Uppercase or Lower Case**
    *   **Outline: Outline Effect (the color of the outline is inherited from the font color). I**f you would like to change the color of the outline you will need to configure additional [color options.](https://support.bynder.com/hc/en-us/articles/8655481718930)
        
5.  **Font Scale** (only available for a text element for a child size, not the main size): The **Font Scale** allows you to adjust the size of linked text elements on child sizes so that they fit the dimensions of the design. Use the slider to adjust the size or manually enter the percentage.
6.  **Color**: Add the text color by manually entering the hex code or click ![](https://support.bynder.com/hc/article_attachments/11686033009042) to select one of the [brand colors](https://support.bynder.com/hc/en-us/articles/8655481718930)
    .
    *   Click **Configure element styles** to allow content creators to change the text color. Check the box next to the brand color that you’d like to make available.
7.  **Alignment**: Set the alignment, which determines how the text is distributed within its bounding box.
8.  **Text wrap:** Click the drop-down to select one of the two ways to wrap text.
    *   **Wrap text:** This option automatically breaks a line of text to fit within the text element's bounding box without adjusting the font size.
    *   **Shrink text to fit:** Automatically adjust the font size on a line of text to fit within the text element bounding box. Control the minimum size of the font as it shrinks by using the slider or manually entering the minimum size percentage.
9.  **Duration** (video only): Set the duration of the text by dragging and dropping the element in the editor or manually entering the duration.
    *   **Timeline behavior**: Click the dropdown to set the timeline behavior. We recommend setting the timeline behavior to **Auto**.
10.  **Visibility control:** Click the toggle to allow creators to hide/show elements during the variation creation process.  
     
    
    ### Note
    
    You can still enable the visibility control if a layer is locked. If it’s locked, the user will be able to hide the element but won’t be able to make any additional edits.
    
11.  **Mask**: Select the mask from the dropdown. Learn more about masking [here](https://support.bynder.com/hc/en-us/articles/6693664394130)
    .
12.  **Effects**: Click the toggle next to the effect to enable it. 
    *    
        *   **Shadow-** Create a shadow effect applied to the text.
        *   [**Backgrounds**](https://support.bynder.com/hc/en-us/articles/20759720480786)
             **-** Create Dynamic Text Backgrounds
13.  **Blend Modes:** Click the dropdown to set the blend mode. Learn more about blend modes [here](https://support.bynder.com/hc/en-us/articles/5072234085394)
    .

    **Q: What is the difference between Studio Basic and Studio Advanced?**
    
    The [Basic](https://support.bynder.com/hc/en-us/articles/20579398218514#h_01J4HRNQD6X7EDB9Z9BS7WZG61)
     package is perfect for those needing essential functionalities to produce high-quality visuals efficiently. For more complex and large-scale projects, the [Advanced package](https://support.bynder.com/hc/en-us/articles/20579398218514#h_01J4HRNQD6RXJECMN03N6WKPFX)
     provides enhanced features such as multi-size designs, AI-translated variations, and batch creation. Select the package that best fits your project's scope and workflow requirements. Learn more [here](https://support.bynder.com/hc/en-us/articles/20579398218514)
    .

    **Q:  Is right-to-left text supported in Studio?**  
    Yes, starting on **October 14, 2024** right-to-left text will be supported.
