# Keyboard Shortcuts in Asset Bank – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013931559-Keyboard-Shortcuts-in-Asset-Bank

See below for a list of keyboard shortcuts you can use in your Asset Bank.

|     |     |
| --- | --- |
| **Shortcut** | **Description** |
| S   | *   [Creates a Shareable Collection](https://support.bynder.com/hc/en-us/articles/15953779016594)<br>     _(works on asset overview page)_ |
| E   | *   Opens up [Editing](https://support.bynder.com/hc/en-us/articles/360013870340)<br>     Window _(works on asset overview page)_ |
| D   | *   Opens up [Download](https://support.bynder.com/hc/en-us/articles/360013931219)<br>     Window _(works on asset overview page)_ |
| C   | *   [Adds Asset to a Collection](https://support.bynder.com/hc/en-us/articles/360013932639)<br>     _(works on asset overview page)_ |
| tab | *   Use the **Tab** key in the upload and edit screen to navigate the available metaproperties and expand their options. If a metaproperty is a text field, the **Tab** key will activate the field so that you can immediately enter text there.<br>*   If you've modified a field and press the **Tab** key to move to another field, your changes will be saved.<br>*   For multi-select options, use **Shift + Tab** to move the highlight up the list and the **Tab** key to move the highlight down the list. |
| shift + tab | *   When you edit an asset from within the Asset Detail View, use the Shift + Tab key combination to move to the previous metaproperty. |
| enter | *   In the Asset detail page, press the Enter key to save modified data. You can also use Enter to select autocomplete options or tags. |
| spacebar | *   Use the spacebar to choose and clear **Select** metaproperty options. |
|     |     |

### Note

The **Tab** key may not work as described in Safari.

To troubleshoot this navigate to **Safari > Preferences > Advanced** and select **Press Tab to highlight each item on a webpage**.
