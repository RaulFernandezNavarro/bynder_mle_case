# View and Download Collections Shared Embedded within Guides and Landing Pages – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/22046924964242-View-and-Download-Collections-Shared-Embedded-within-Guides-and-Landing-Pages

When a [Guide or Landing Page](https://support.bynder.com/hc/en-us/articles/7382396334482)
 includes a shared Collection, you can easily browse and download its assets, provided the [permissions](https://support.bynder.com/hc/en-us/articles/7382396334482)
 allow it.

If you are a Bynder user, please log in to maximize your access to individual asset downloads.

How to View a Collection[](https://support.bynder.com/hc/en-us/articles/22046924964242-View-and-Download-Collections-Shared-Embedded-within-Guides-and-Landing-Pages#h_01JADTPQK97M5WA5P8H7XMCTJF)

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1.  To see all the assets in the Collection, select the last thumbnail of the preview. 
2.  This will expand the view to display all the assets available within the Collection.
3.  If you want to reduce the view and only see the default preview assets, click the Show Less button. This will condense the display to the initially configured view.
4.  (If applicable) you can preview individual assets (if applicable) by clicking on a specific asset within the Collection. This will open and preview it within the Asset Bank.

How to Download a Collection[](https://support.bynder.com/hc/en-us/articles/22046924964242-View-and-Download-Collections-Shared-Embedded-within-Guides-and-Landing-Pages#h_01JADTT1MCFMCZ4T8FWKN0V5F7)

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1.  If available, click the **Download** **Collection** button.
2.  This will package all the assets, including their relevant derivatives (if any), into a ZIP file for easy download.
3.  If available, you can download specific assets by hovering over an individual asset within the Collection and clicking the **Download** **Asset** button.
4.  If the asset has derivatives, they will be included in a ZIP file. For single-file formats, such as PDFs, the file will download directly.
