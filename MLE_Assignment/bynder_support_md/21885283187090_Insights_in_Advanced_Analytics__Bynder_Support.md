# Insights in Advanced Analytics – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/21885283187090-Insights-in-Advanced-Analytics

The Insights feature in [Advanced Analytics](https://support.bynder.com/hc/en-us/articles/14829008886930)
 offers predefined, digestible, and actionable insights that highlight areas needing attention or reveal interesting trends in your DAM. Continuously identifying issues and opportunities provides proactive recommendations for admins, content optimization, and user adoption.

With quick access to these insights, you can take immediate corrective actions, uncover hidden trends for informed decision-making, and ensure long-term DAM health through regular cleanup and security monitoring. Additionally, it offers guidance on asking the right questions to optimize your DAM efficiently.

**Who can view analytics reports?**

Users with the following permissions can access and download reports:

*   [**View advanced reporting**](https://support.bynder.com/hc/en-us/articles/18433246503442-Permissions-in-Asset-Bank#:~:text=View%20advanced%20reporting%20%2D%20Allows%20users%20to%20access%20Bynder%20Analytics.)
    

How to Access Insights[](https://support.bynder.com/hc/en-us/articles/21885283187090-Insights-in-Advanced-Analytics#h_01J9S8HT9KAX9P09RTM3N9K4JV)

1.  Navigate to **Settings** > **Reporting** > **Analytics**.
2.  The **Insights** section will automatically appear.![](https://support.bynder.com/hc/article_attachments/21885283184274)
3.  View the latest [available insights.](https://support.bynder.com/hc/en-us/articles/21885283187090-Insights-in-Advanced-Analytics#h_01J9S8NB2TH554ZYMVB6HV2YZN)
     

Available Insights[](https://support.bynder.com/hc/en-us/articles/21885283187090-Insights-in-Advanced-Analytics#h_01J9S8NB2TH554ZYMVB6HV2YZN)

----------------------------------------------------------------------------------------------------------------------------------------------

Some insights may not be visible in your portal unless the trigger has been met. 

|     |     |     |
| --- | --- | --- |
| **Title** | **Description** | **Trigger** |
| \# Assets | Have not been downloaded in the past 180 days. | Number of Assets > 0 |
| \# Assets | Uploaded assets (past 30 days) without any downloads. | Number of Assets > 0 |
| User Name | Users with no downloads in the last 30 days but activity in the previous 60 days. | Number of Users > 0 |
| \# Assets | Downloaded in the last 7 days, -/+ % from previous period | Always |
| \# Assets | Uploaded in the last 7 days, -/+ % from previous period | Always |
| \# Assets | Downloaded in the last 7 days -/+ % from previous period | Always |
| \# Assets | Schedule to Archive within 7 days with X recent downloads | Number of Assets > 0 |
| \# Assets | Schedule to Lose Limited Usage status within 7 days with X recent downloads | Number of Assets > 0 |
| Asset Name | Is the top Asset with # Downloads in the last 7 days | Always |
