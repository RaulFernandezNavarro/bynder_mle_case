# Set Up Webhooks in Content Workflow for Notifications – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/19596398706322

Webhooks can be configured in Content Workflow to receive notifications for various events. By setting up webhooks, you can specify actions that will activate the webhook to send a notification to a specific endpoint. This helps update your stakeholders about significant events or modifications in Content Workflow.

How to Set up a Webhook[](https://support.bynder.com/hc/en-us/articles/19596398706322#h_01J0KNDQG5J2EX6QVM9PPJ9V7K)

--------------------------------------------------------------------------------------------------------------------

1.  Navigate to the project and select the **Integrations** tab.![[CW] Integrations Tab.png](https://support.bynder.com/hc/article_attachments/19599098686610)
2.  Select the **Webhooks** tab.![[CW] Webhook Tab.png](https://support.bynder.com/hc/article_attachments/19599213796882)
3.  To add a webhook enter the webhook URL, and select the event you wish to subscribe to. Complete the action, by clicking **Create new webhook**.![[CW] Create New Webhook.png](https://support.bynder.com/hc/article_attachments/19599213798930)
4.  Once added, the webhook will send a JSON payload of data whenever the subscribed to event is fired.

Existing Webhooks[](https://support.bynder.com/hc/en-us/articles/19596398706322#h_01J0KY7HX53Z4BRVGJQZT4S52F)

--------------------------------------------------------------------------------------------------------------

Any of your existing webhooks for the project will be listed under **Existing** **webhooks.**

Webhook Payloads[](https://support.bynder.com/hc/en-us/articles/19596398706322#h_01J0KY801P4V3XYW0CXRKCDQBT)
