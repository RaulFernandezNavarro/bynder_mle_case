# Studio Analytics Dashboard in Advanced Analytics – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/17849820547602-Studio-Analytics-Dashboard-in-Advanced-Analytics

Gain insight into how often users log into [Studio](https://support.bynder.com/hc/en-us/articles/360014321859)
, create and export designs, and use templates and variations. These metrics can help improve content creation, track Studio usage and adoption, and understand how your team is using the solution. 

[Schedule reports and dashboards](https://support.bynder.com/hc/en-us/articles/17406736018066)
 and [create alerts](https://support.bynder.com/hc/en-us/articles/17405894062994)
 to be sent directly to users' and non-users' emails for higher visibility into how your portal's performance and usage. 

You must have [Advanced Analytics](https://support.bynder.com/hc/en-us/articles/14829008886930)
 and [Studio](https://support.bynder.com/hc/en-us/articles/360014321859)
 enabled in your portal in order to see the Studio Dashboards.

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

**Who can view analytics reports?**

Users with the following permissions can access and download reports:

*   [**View advanced reporting**](https://support.bynder.com/hc/en-us/articles/18433246503442-Permissions-in-Asset-Bank#:~:text=View%20advanced%20reporting%20%2D%20Allows%20users%20to%20access%20Bynder%20Analytics.)
    

Studio Glossary[](https://support.bynder.com/hc/en-us/articles/17849820547602-Studio-Analytics-Dashboard-in-Advanced-Analytics#h_01HV9GVX9CVND68ZJSG59BXXV0)

-------------------------------------------------------------------------------------------------------------------------------------------------------------

**Designs**: can be created in three ways: from scratch, by importing them, or based on a Template. Studio Admins and Content Designers can create designs from scratch or import them, while Content Creators can only create designs using pre-existing templates.

**Templates:** are "master" files created by Studio Admins or Content Designers that can be reused and customized by Content Creators to create their own designs. Ready-made templates are also available. 

**Exports:** refer to downloading a design or template in Studio or exporting to the Waiting Room.

**Variations**: All Studio users can create variations, which can be created from scratch or imported. They are versions of a design with a slight change such as translated text.
