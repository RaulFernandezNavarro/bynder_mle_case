# Assign Edit Rights to Stages in Asset Workflow – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013871420-Assign-Edit-Rights-to-Stages-in-Asset-Workflow

You can now decide who can edit stages in the workflow and who can only view them. The edit rights include the rights to submit the workflow to the next stage, add and remove tasks, assets or attachments, or upload versions of assets.

![editable-by.png](https://support.bynder.com/hc/article_attachments/10108141344914)

### Note

Users can only edit active stages. Users who can edit stages can delete all assets linked to the workflow, not only their own assets. Submitting a job to the next stage still requires the **Approve jobs** permission.

Before you begin[](https://support.bynder.com/hc/en-us/articles/360013871420-Assign-Edit-Rights-to-Stages-in-Asset-Workflow#h_01HP4TZFRSQEEBAGYTNHV50HCM)

----------------------------------------------------------------------------------------------------------------------------------------------------------

Make sure your users have the rights to edit stages. If users do not have the right to edit stages, they will not be able to edit the stage even though you have selected them in the Editable by field.

1.  Go to **Settings > Users & rights > Permission Management**.
    
2.  Select the profile which you want to be able to edit stages.
    
3.  Enable the **Manage stages** permission.
    
4.  Click **Save** to save the changes.
    

 [](https://support.bynder.com/hc/en-us/articles/360013871420-Assign-Edit-Rights-to-Stages-in-Asset-Workflow#h_01HP4TZFRSVHWCTS9N2SWRY4YV)

-------------------------------------------------------------------------------------------------------------------------------------------

1.  Go to **Settings > Workflow > Preset Management**.
    
2.  Click **Job**.
    
3.  Click the preset in which you want to edit the stages.
    
4.  For every stage, select the right option from the **Editable by** drop-down:
    
    *   **All** - all users that have the edit rights in their profile can edit the stage.
        
    *   **Stage responsibles** - only the person set as the person responsible for this particular stage can edit it.
        
    *   **Stage responsibles & job accountable** - only the person set as the person responsible for this particular stage and the person accountable for the entire job can edit the stage.
        
    
    ### Note
    
    If you do not select anything in the drop-down, the default value is All.
    
5.  Save your changes.
    
    *   If you want to update future jobs, click **Save & close**.
        
    
    ### Note
    
    If you grant the edit rights to a specific person or group, you can still edit the stage if you have the **Get all available workflow permissions** (admin only) permission enabled for your profile.
