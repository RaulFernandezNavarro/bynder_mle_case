------------------------------------------------------------------------------------------------------------------------------------------------------

Users can download one or more assets stored in the Asset Bank. Your portal admin might ask you to provide a reason for downloading an asset via a form. 

Who? [](https://support.bynder.com/hc/en-us/articles/360013931219-How-To-Select-And-Download-Assets-In-The-Asset-Bank#h_01HJ92KGN62JM6R395EGFTX1E8)

----------------------------------------------------------------------------------------------------------------------------------------------------

This feature/solution is enabled by a Bynder Admin.

As a Bynder Admin, you can manage who can download certain asset types. Users with the _Download original assets_ permission can download the original files and derivatives of public assets. Users with the _Export metadata CSV file on download_ can download assets with metadata. Users with the _Download non-public asset derivatives_ permission can download non-public derivatives.

How?[](https://support.bynder.com/hc/en-us/articles/360013931219-How-To-Select-And-Download-Assets-In-The-Asset-Bank#h_01JXQ7QXEMCNW209RWH257YYR4)

### Selecting Assets

1.  Navigate to your **Asset Bank.**
2.  **Select** the assets by clicking on the asset. A selected asset will have a colored checkmark in the top righthand corner of the preview.![Screenshot 2023-12-01 at 11.10.26 AM.png](https://support.bynder.com/hc/article_attachments/27401102904722)
    *   To select multiple assets, click the **left** **button** on your mouse and drag the mouse over the assets you want to select. When you release the mouse button, the items in the blue selection will be selected.
3.  Expand the arrow in the top filter bar to **Select all,** **Select** **visible** **media**, or **Select** **none**.![Screenshot 2023-12-01 at 10.48.27 AM.png](https://support.bynder.com/hc/article_attachments/27401102910738)
    *   See how many assets have been selected by looking at the selected asset count in the top right corner.

### Downloading a Single Asset

1.  Navigate to your **Asset Bank.**
2.  **Search** or filter assets to find the one you want.
3.  **Select** the asset you would like to download.
4.  In the Asset Detail View, scroll down to **Download all files.**
5.  **Click** one of the available download buttons in this section based on your needs. The download will start automatically and will be **saved** locally.
    *   By default, downloads contain the original file name.
    *   If you want the asset title listed as the file name, prefixes, and suffixes attached, you may contact Customer Support to configure this within your portal.
    *   On specific operating systems and web browsers, if the filename contains certain special characters (\*?&lt;&gt;~), they might be replaced by an underscore in the downloaded asset's filename.

### Downloading Multiple Assets

1.  Navigate to your **Asset Bank.**
2.  **Search** or filter assets to find the one you want.
3.  **Select** the assets you wish to download, or you can click **Command-A** to select all the assets. 
    *   We would recommend that you download in batches. When triggering the download, the system has to zip the file first. The bigger the file is (the more files you have), the more likely it is to fail, as you might have a time-out, or lose internet connection.
4.  Click the ![download.png](https://support.bynder.com/hc/article_attachments/15950522115346) button in the filter bar. ![Screenshot 2023-12-22 at 10.17.05 AM.png](https://support.bynder.com/hc/article_attachments/15950522125714)
5.  The **Download selection** pop-up window will open, and you can select from the list of download options.
6.  Click **Start Download.** Your download will be saved locally.
7.  By default, downloads contain the original file name.
    *   Contact Customer Support to configure your portal if you want the asset title listed as the file name, prefixes, and suffixes attached or to customize your file name.
        *   The downloaded files will appear in the following formats:
            *   Prefix-original filename. extension
            *   original filename-suffix.extension
            *   prefix-original filename-suffix. extension
            *   (Optional) Notify a user via email when their download is ready. They will receive an email with the download link. Contact Customer Support to enable this feature in your portal. 
            *   On specific operating systems and web browsers, if the filename contains certain special characters (\*?&lt;&gt;~), they might be replaced by an underscore in the downloaded asset's filename.

**When will a zip file be created?**

A Zip file will be created if you select more than one asset (regardless of size).
