# Users Dashboard in Advanced Analytics – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/20094187625490-Users-Dashboard-in-Advanced-Analytics

The Users Dashboard in [Advanced Analytics](https://support.bynder.com/hc/en-us/articles/14829008886930)
 provides a powerful, in-depth overview of user-related metrics, enabling administrators to track onboarding progress, analyze user behavior, and optimize platform engagement. This dashboard allows you to measure the return on investment (ROI) of your Bynder platform, improve adoption rates by addressing inactive user groups, and ensure successful onboarding for new users. With insights like Login Trends Over Time and New Users by User Group Over Time, admins can monitor engagement, identify areas for improvement, and take targeted actions such as training or resource allocation.

By leveraging these dashboards, administrators, and stakeholders can gain valuable insights into user behavior, optimize platform usage, and enhance overall user experience within the Bynder portal.

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

The revamped Users Dashboard solves challenges related to user activity visibility, enabling admins to:

*   Understand how many users are actively logging in and which user groups they belong to.
*   Analyze the behavior of newly onboarded users, such as logins and value-generating actions (e.g., downloads).
*   Exclude logins from removed users for more accurate reporting.

Overview Metrics[](https://support.bynder.com/hc/en-us/articles/20094187625490-Users-Dashboard-in-Advanced-Analytics#h_01JDJK2FF6MWBC0J9G7K27B05N)

---------------------------------------------------------------------------------------------------------------------------------------------------

These top-level metrics provide a snapshot of user activity:

*   Total Users: Displays the total registered users in the portal, distinguishing between active and inactive accounts.
*   New Users: Tracks newly created user accounts to monitor onboarding trends.
*   Logins: Measures the total number of logins within a specified timeframe.
*   Users with Logins/Downloads: Highlights users who have engaged with the portal through logins or downloads.
*   Login % and Download %: Reflects engagement rates relative to total users.  
    Average Logins per User: Offers insights into how frequently users engage with the portal.

Track user activity by organizational structures such as departments or teams:

*   User Groups: Lists all defined user groups within the portal.
*   User Group Breakdown: Compares login activity against total users in each group.
*   Login Distribution: Visualizes login frequency across groups to identify active and less active departments.
*   User Group Activity Trends: Helps target specific groups for training or resources.
