# Duplicate Finder in the Waiting Room – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/13811974882962

The duplicate finder in the [waiting room](https://support.bynder.com/hc/en-us/articles/360013870280)
 allows you to check for 100% true duplicate assets before they are approved and uploaded to the Asset Bank. This feature helps you easily manage incoming assets and ensure that unnecessary duplicates are not added to the portal. All [Bynder-supported file types](https://support.bynder.com/hc/en-us/articles/360013936359)
 are scanned for duplicates (image, document, and video types). 

Learn more about finding duplicates [in the Asset Bank](https://support.bynder.com/hc/en-us/articles/360013870020)
 or [during upload](https://support.bynder.com/hc/en-us/articles/13812491746194-Find-duplicates-during-upload)
.

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Check for Duplicates in the Waiting Room[](https://support.bynder.com/hc/en-us/articles/13811974882962#h_01HAN98CR52PMHDXC10G49DNA8)

-------------------------------------------------------------------------------------------------------------------------------------

1.  Navigate to ![](https://support.bynder.com/hc/article_attachments/13815743828498) **Settings** > **Waiting Room.**
2.  Navigate to the **Request Type** that you’d like to audit, then select a request.
3.  Click **Check for Duplicates** in the top right corner. ![01 Screenshot 2023-11-18 at 04.39.28.png](https://support.bynder.com/hc/article_attachments/15211560760082)
4.  If duplicates exist, ![](https://lh6.googleusercontent.com/-hTf8cOFt2XEsMdKGyLuT8S6fWBnhfwGH3xUgjZdILOnLVP_AtJXVU9lMWhHsKzFQHDvRehFs3n_ZjIjSxtyfHB4XhpiPtwev2KdVPDbE5CuwAYZdv7_1a_yH72cJCqt7AMWG7HCApdlM4a1CDZJ5Vk) will display on the asset thumbnail. ![03 Screenshot 2023-11-18 at 04.39.53.png](https://support.bynder.com/hc/article_attachments/15211556037778)  
    ![02 Screenshot 2023-11-18 at 04.39.48.png](https://support.bynder.com/hc/article_attachments/15211556042258)
5.  If no duplicates exist, a **No duplicate(s) found** message will display.  
    ![](https://support.bynder.com/hc/article_attachments/13815727634066)

Important Information[](https://support.bynder.com/hc/en-us/articles/13811974882962#h_01HENDEP08B1WJG48SB4A6R2TN)

------------------------------------------------------------------------------------------------------------------

*   Uploads that are cropped, are in different file formats or have different text that appears in the image will not be flagged as duplicates but as [Similar Assets.](https://support.bynder.com/hc/en-us/articles/13811681142802)
       
      
    
*   Metadata embedded into the file itself is also considered when checking for duplicates. If there are differences, the images will not be picked up. For example, two identical photos, one with embedded GPS coordinates and the other not having them, will not be considered duplicates.
*   ### Note
    
    Some of these features require an API user to be created and added to a specific [permission profile](https://support.bynder.com/hc/en-us/articles/10309197472914)
     to **view** **all** **assets** (limited and archived included). This user should always be retained as it will affect the ability to use these features. (This applies to the [Duplicate Finder During Upload](https://support.bynder.com/hc/en-us/articles/13812491746194)
     and  [Duplicate Finder in the Waiting Room](https://support.bynder.com/hc/en-us/articles/13811974882962)
     features[.](https://support.bynder.com/hc/en-us/articles/13811974882962)
