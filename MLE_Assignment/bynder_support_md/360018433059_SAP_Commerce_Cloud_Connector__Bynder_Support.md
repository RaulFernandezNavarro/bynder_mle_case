After configuration, you can manually add Bynder assets to products and CMS pages through Backoffice and SmartEdit. In addition, you can search, browse, and filter Bynder assets within SAP Commerce Cloud. When uploading a new asset to Bynder, you can tag it with a unique identifier linked to a product in SAP. For example, a product ID. This way, the connector can automatically sync the asset with the corresponding product in SAP.  The information coming from SAP can be used to tag your Bynder metaproperties with the correct data automatically. Optionally, you can set up a Bynder metaproperty that indicates for which SAP multiple media attribute fields this asset needs to be used. By selecting the appropriate metaproperty options upon upload to Bynder, you can automatically ensure that the asset will be used for the appropriate SAP multiple media attribute fields.

1.  Log in to the Backoffice of your SAP Commerce environment.
    
2.  Go to **Catalog > Products** in the left sidebar.
    
3.  Open the product you want to add a product image to.
    
4.  Click the **Multimedia** tab. In the **Images** section, you'll find the available image variations.
    
5.  Click the box of the image variation you want to associate a Bynder asset to.
    
6.  Click the button, which will open the [Compact View](https://support.bynder.com/hc/articles/360014369640#UUID-eb5b394d-4148-0471-d3bb-cabb368ca003)
    .
    
7.  Click **Connect** if your company's Bynder portal URL is already filled in in the popup. If not, enter the URL in the below format. A popup will open, which allows you to log in to your Bynder environment. Log in to the portal the usual way.
    
    https://YourPortalURL.com
    
8.  (Optional) You can use the available search bar and filters to find the image you want to add.
    
9.  You can just select the image you want to add.
    
10.  Click the **Add asset** button to add the image to your Content Builder content. The Compact View will close.
    
11.  Open the **Select Image Size** dropdown and select whether you want to add the original file or one of the available derivatives.
    
12.  Click **Add Asset** to add the image to the product. You can close the Edit Item screen.
    

### Note

*   You can only select **Original** if the asset is marked as public.
    
*   The dropdown will only show the public derivatives.
    

Add Bynder Images to a Storefront Page in SmartEdit[](https://support.bynder.com/hc/en-us/articles/360018433059-SAP-Commerce-Cloud-Connector#h_01HDKREHSXPR3FXBE1GA42PA88)

1.  Log in to the SmartEdit section of your SAP Commerce environment.
    
2.  Go to **Pages** in the top menu and open the page you want to add a Bynder image to.
    
3.  Enable the **Advanced Edit** mode in the toolbar at the top of the page.
    
4.  Add a new image component or edit an existing image.
    
5.  Click the button. A new screen will open. You can use the search bar to search for the asset you want.
    
6.  Select the image you want to add.
    
7.  Open the **Select your image** dropdown and select whether you want to add the original file or one of the available derivatives.
    
8.  Click **Add asset** in the top right corner.
    
9.  Click the **Upload** button in the **Media** section.
    
10.  Ensure all the required fields are filled in, and click **Save** to add the image to your content.
    

### Note

*   You can only select **Original** if the asset is marked as public.
    
*   The dropdown will only show the public derivatives.
