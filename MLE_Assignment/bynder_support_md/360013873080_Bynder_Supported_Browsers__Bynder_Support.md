# Bynder Supported Browsers – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013873080-Bynder-Supported-Browsers

Bynder works with popular web browsers. For the best experience, we recommend using Google Chrome and keeping it up to date with the latest version.

We support the following browsers:

*   Google Chrome (latest stable version)
*   Microsoft Edge (latest stable version)
    
*   Firefox (latest stable version)
    
*   Safari (latest stable version)
    

_Note: Mobile versions of browsers are not supported, we cannot guarantee that Bynder will work when using them._
