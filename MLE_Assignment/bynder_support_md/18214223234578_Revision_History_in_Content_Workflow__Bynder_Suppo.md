# Revision History in Content Workflow – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/18214223234578

Revision History in Content Workflow simplifies version control and collaboration by allowing users to [compare](https://support.bynder.com/hc/en-us/articles/18214223234578#h_01JDAETZC8QC2Z2B9RWCGP1FYA)
 parent revision entries, track edits, [restore](https://support.bynder.com/hc/en-us/articles/18214223234578#h_01JDAEXGSMHMNWF38SSXBMRBQX)
 specific fields, and [duplicate](https://support.bynder.com/hc/en-us/articles/18282325852562)
 entire items from previous versions. The feature highlights changes in green for additions and red for deletions, streamlining auditing and ensuring transparency in the content creation process.

How to Access Revision History[](https://support.bynder.com/hc/en-us/articles/18214223234578#01JDAF3562VCQPA7HJG28K3HJN)

-------------------------------------------------------------------------------------------------------------------------

### For the Entire Item

1.  Click ![Screenshot 2024-04-09 at 3.43.12 PM.png](https://support.bynder.com/hc/article_attachments/18214239197970)in the top-right corner of the item.
2.  Select Revision History from the dropdown menu.
3.  This will display the revision history for the entire item.

### For an Individual Field

1.  Locate the field for which you want to view the revision history.
2.  Click![Screenshot 2024-04-09 at 3.43.12 PM.png](https://support.bynder.com/hc/article_attachments/18214239197970)next to the field.
3.  Select **Revision** **History** to view historical changes specific to that field.

Exploring Revision History[](https://support.bynder.com/hc/en-us/articles/18214223234578#01HV26VS2FTQD460P32DAKVYH9)

---------------------------------------------------------------------------------------------------------------------

When viewing the revision history, the layout is divided into two sections:

*   Left Side: Displays the content as it appears in each version.
*   Right Side: Lists all available revision entries, including the date, time, and user who made the changes.

How to Compare Content Changes[](https://support.bynder.com/hc/en-us/articles/18214223234578#h_01JDAETZC8QC2Z2B9RWCGP1FYA)

---------------------------------------------------------------------------------------------------------------------------

1.  Navigate to the **Revision** **History** view for the item or field.
2.  On the right-hand side, select a revision entry to review its content.
3.  To compare this version with another:
4.  Select the first revision you want to compare.
5.  Select a second revision.
6.  Select **![](https://support.bynder.com/hc/article_attachments/22809032989074)**
7.  Once the two versions are selected, differences will automatically appear in the content view.
8.  Differences between the two versions are highlighted:
    *   **Red**: Content that was removed.
    *   **Green**: Content that was added.
    *   **Purple**: Content where the formatting has been changed (e.g. Bold)
9.  To exit comparison mode, hover over a revision entry and select **stop** **comparing** or another revision entry.

How to Restore Content[](https://support.bynder.com/hc/en-us/articles/18214223234578#h_01JDAEXGSMHMNWF38SSXBMRBQX)

-------------------------------------------------------------------------------------------------------------------

You cannot restore an item if the status is locked or in read-only mode.

### Restoring Content in a Field

1.  In the **Revision** **History** view for the field.
2.  Select the version you want to restore.
3.  Click the **Restore** button.
4.  The content from that version will overwrite the current field content.

### Restoring an Entire Item

1.  Access the **Revision** **History** for the whole item.
2.  Select the desired version.
3.  Instead of overwriting the current item, the system duplicates the selected version into a new item for you to edit.
