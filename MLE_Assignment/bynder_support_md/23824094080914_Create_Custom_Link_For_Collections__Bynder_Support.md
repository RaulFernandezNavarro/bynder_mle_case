# Create Custom Link For Collections – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/23824094080914

[Custom links](https://support.bynder.com/hc/en-us/articles/21861490210450)
 simplify [sharing Collections](https://support.bynder.com/hc/en-us/articles/15953779016594)
 by providing branded, vanity URLs, making asset sharing more streamlined and professional. This feature enhances the user experience, strengthens brand identity, and improves the sharing process for both internal and external stakeholders. 

How to Enable This Feature[](https://support.bynder.com/hc/en-us/articles/23824094080914#h_01JH69RYQZWTGK9X52ZWAP3YS7)

-----------------------------------------------------------------------------------------------------------------------

Custom links are available to all Bynder users with the [appropriate permissions.](https://support.bynder.com/hc/en-us/articles/23824094080914#h_01JAQNC8KNV892F78N8SQ2J63X)

[Custom Links Permissions](https://support.bynder.com/hc/en-us/articles/18433250182802-Permissions-in-CX-User-Community-Brand-Guidelines#:~:text=the%20respective%20solution.-,Custom%20Links%20Permissions,-Create%20Custom%20Links)
[](https://support.bynder.com/hc/en-us/articles/23824094080914#h_01JAQNC8KNV892F78N8SQ2J63X)

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

### Create Custom Links

This permission allows users to [create custom links](https://support.bynder.com/hc/en-us/articles/21861490210450)

### Manage Custom Links

Users with this permission can [edit and delete](https://support.bynder.com/hc/en-us/articles/21861490210450)
 all custom links, including those created by others.

### View Custom Links

Users with this permission can **view** all custom links, including those created by other users.

Before You Begin[](https://support.bynder.com/hc/en-us/articles/23824094080914#h_01JH69TCFBDFE86A5DA340A2X6)

-------------------------------------------------------------------------------------------------------------

Before [creating a custom link](https://support.bynder.com/hc/en-us/articles/23824094080914#h_01JH69ZT6EJX04W7WG0Q98GN3G)
, ensure the collection’s **link-sharing option** is enabled. Without this, custom link creation will not be possible.

How to Create a Custom Link for Collections[](https://support.bynder.com/hc/en-us/articles/23824094080914#h_01JH69ZT6EJX04W7WG0Q98GN3G)

----------------------------------------------------------------------------------------------------------------------------------------

1.  Navigate to the [collection](https://support.bynder.com/hc/en-us/articles/360013932639-Create-Collections)
     you want to share.
2.  Click [**Share Collection.![Screenshot 2025-01-09 at 3.04.36 PM.png](https://support.bynder.com/hc/article_attachments/23824392540818)**](https://support.bynder.com/hc/en-us/articles/15953779016594-Share-Collections)
    
3.  In the sharing menu, select the option to **Create Custom Link.**
4.  If the collection is already [published](https://support.bynder.com/hc/en-us/articles/15954306339474-Create-a-Public-Collection#:~:text=Create%20a%20Public%20Collection%C2%A0)
     online, you can **create a custom link** to that published link directly.
    *   If not, a notification will prompt you to publish it online first.![Screenshot 2025-01-09 at 3.05.38 PM.png](https://support.bynder.com/hc/article_attachments/23824410998930)
5.  **Customize** the URL as needed. Set [duration](https://support.bynder.com/hc/en-us/articles/23824094080914#h_01JD2Q3NHANR02883EW8HD3RQ8)
    , [password](https://support.bynder.com/hc/en-us/articles/23824094080914#01JD2Q90CA4RJ56BYHWZGYAJ11)
    , or [QR code.](https://support.bynder.com/hc/en-us/articles/23824094080914#01JD2QHNH4GYH9B4BTCHY8BMPE)
     
6.  Click **Save**.![Screenshot 2025-01-09 at 3.06.30 PM.png](https://support.bynder.com/hc/article_attachments/23824392543250)
7.  You can now share the link with users.

Set Duration (optional)[](https://support.bynder.com/hc/en-us/articles/23824094080914#h_01JD2Q3NHANR02883EW8HD3RQ8)

--------------------------------------------------------------------------------------------------------------------

1.  Navigate to the **Duration** **tab** and enable the **Set** **Duration** option to set the link's start and end date.
2.  Define the link's start and end dates in the **Start** **date** and **End** **date** fields.  
    *   Links will not work before the start date or after the end date.
3.  You can modify the duration after setting it.
    *   If you choose to disable this toggle the duration **will no longer apply,** and the link will remain active indefinitely.

![](https://support.bynder.com/hc/article_attachments/23824208939154)

Set Password (optional)[](https://support.bynder.com/hc/en-us/articles/23824094080914#01JD2Q90CA4RJ56BYHWZGYAJ11)

------------------------------------------------------------------------------------------------------------------

1.  Navigate to the **Password** tab and enable the **Set Password option**. 
2.  Ensure the password is at least six characters long.
3.  Enter the password in the **Enter** **password** and **Re-enter password f**ields.
    *   Toggling off the **Set** **Password** **option** removes password protection.

![](https://support.bynder.com/hc/article_attachments/23824207172242)

### Note

*   Passwords are not displayed after creation and protect access **only via the custom link**.
*   Public Guides and other content remain accessible through the original URL.

Customize QR Code (optional)[](https://support.bynder.com/hc/en-us/articles/23824094080914#01JD2QHNH4GYH9B4BTCHY8BMPE)

-----------------------------------------------------------------------------------------------------------------------

This allows users to access the custom link by scanning the QR code with their device. A QR code is automatically generated and can be customized. 

1.  Navigate to **QR** **code** tab.
2.  (Optional) Enable the **Logo** option to display your account logo at the center of the QR code
3.  (Optional) Use the **Foreground** **color** and **Background** **color** fields to pick colors with the color picker or by entering a hex code. icon.
4.  (Optional) Click **Select** **logo** to choose a different image for the QR code logo.![](https://support.bynder.com/hc/article_attachments/23824208941586)

How to Edit Custom Links[](https://support.bynder.com/hc/en-us/articles/23824094080914#h_01J9PP1DF8XDHPZ9B6PYBR6D6G)

---------------------------------------------------------------------------------------------------------------------

1.  Select ![](https://support.bynder.com/hc/article_attachments/23824207174418) next to the custom link you want to edit, click **Edit**.
2.  Make the necessary changes in the available tabs.
3.  Select **Update** to save the changes.

![](https://support.bynder.com/hc/article_attachments/23824208942866)How to Delete Custom Links

1.  Select ![](https://support.bynder.com/hc/article_attachments/23824207174418)  next to the custom link you want to remove and click **Delete**.
2.  Please read the warning message carefully and click **Yes**, delete to permanently remove the custom link and all associated analytics.
3.  This action is permanent and can't be undone. ![](https://support.bynder.com/hc/article_attachments/23824208943250)
