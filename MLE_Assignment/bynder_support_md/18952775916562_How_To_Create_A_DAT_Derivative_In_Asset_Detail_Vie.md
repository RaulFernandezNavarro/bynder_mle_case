# How To Create A DAT Derivative In Asset Detail View – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/18952775916562

Summary[](https://support.bynder.com/hc/en-us/articles/18952775916562#h_01JKSYT80QTWJCVY3BQSFB2YBF)

----------------------------------------------------------------------------------------------------

You can create Dynamic Asset Transformations(DAT) on the fly. DAT allows you to generate tailored versions of assets, called derivatives, for different systems by adding specific parameters to the DAT derivative URL. Using Dynamic Asset Transformations suited to the platform or device your users use ensures an optimal user experience.

Who?[](https://support.bynder.com/hc/en-us/articles/18952775916562#h_01JKT1WHBCKXBCWKYWRY8S6HAT)

-------------------------------------------------------------------------------------------------

This feature/solution requires your Customer Success Contact to enable, but then individual permissions can be done by the Bynder Admin.

Please keep in mind that in order to have all DAT functionality your Bynder Admin must enable all DAT specific permissions in the DAT section of Permission profiles. Once the permission profile is set up properly all users with DAT permissions can use the feature.

Why?[](https://support.bynder.com/hc/en-us/articles/18952775916562#h_01JKT270WGYF05H21X0C5EAJY7)

-------------------------------------------------------------------------------------------------

DAT is a powerful tool you can add onto your Bynder package, allowing you to create asset derivatives in your asset bank quickly and easily. 

How?[](https://support.bynder.com/hc/en-us/articles/18952775916562#h_01J99RNTB1VWN6A7D0599FMGD0)

-------------------------------------------------------------------------------------------------

1.  Navigate to the **Asset Bank**.
    
2.  Select the asset you want to create a DAT derivative for.
    
3.  Click **Transformations** in the **Info** tab of the _Asset Detail View_. This will open a screen to configure the DAT transformations.
    
4.  Either click the DAT preset you want to apply in the right sidebar, where a preview of what the DAT derivative will look like will be generated, or we can create a derivative from scratch.
    
5.  To create a derivative from scratch click the ![transform.png](https://support.bynder.com/hc/article_attachments/10641651213842) button, and select **Transform**.![Screenshot 2024-05-22 at 3.05.36 PM.png](https://support.bynder.com/hc/article_attachments/19091490419218)
    
6.  Click the **➕ Add transformation** button.
    
7.  Open the **Transformations** dropdown and select one of the available transformations.  ![Screenshot 2024-05-22 at 3.06.10 PM.png](https://support.bynder.com/hc/article_attachments/19091506728850)
    
8.  Choose the parameters for the fields, sliders, and options to configure the transformation.
    
9.  If you’d like to update the focus point click the dropdown and select **Override** **focus** **point**.
    *   You can select a new focus point by editing the percentages or directly clicking on the asset with the real-time editor. This focus point will only be applied to the transformed asset; it won’t change the preset DAM focus point.![Screenshot 2024-05-22 at 3.06.43 PM.png](https://support.bynder.com/hc/article_attachments/19091490430866)
10.  Click the **Format** dropdown and select the **file extension** type you want the output file of your DAT derivative to have. Options are: WebP, JPG, PNG, AVIF, or Auto (recommended).![Screenshot 2024-05-22 at 3.08.31 PM.png](https://support.bynder.com/hc/article_attachments/19091490441234)
    

_Note: If you select **Auto**, WebP will be the default output for all browsers that accept WebP; in all other cases, it will be JPG._

11\. Click the **Quality** dropdown and select one of the available options to set the quality of the output.![Screenshot 2024-05-22 at 3.08.22 PM.png](https://support.bynder.com/hc/article_attachments/19091506741266)

12\. Click the ![link_brand_guidelines.png](https://support.bynder.com/hc/article_attachments/10641651231890) button to copy the URL of the DAT derivative, or select **Download** to download the file. 

13\. Click **Done** to close the screen.![Screenshot 2024-05-22 at 3.07.27 PM.png](https://support.bynder.com/hc/article_attachments/19091506747666)
