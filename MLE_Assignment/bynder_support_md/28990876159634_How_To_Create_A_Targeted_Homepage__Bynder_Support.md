# How To Create A Targeted Homepage – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/28990876159634

Summary[](https://support.bynder.com/hc/en-us/articles/28990876159634#h_01K3G8YY7K2AJPV8JQ0GKA7WDN)

----------------------------------------------------------------------------------------------------

Bynder understands that some users want to show different homepage content to different audiences, in order to make this possible Bynder offers **Targeted Homepages** allowing users to create personalized experiences within the DAM.

Who?[](https://support.bynder.com/hc/en-us/articles/28990876159634#h_01K3G944TC25A1AQF35Y050CEC)

-------------------------------------------------------------------------------------------------

This feature/solution is enable-able by any Bynder portal user.

Users who should be able to create or edit a Homepage need the _Manage Homepage_ permission. 

Why?[](https://support.bynder.com/hc/en-us/articles/28990876159634#h_01K3G958Q23C080Y21BBBT45PT)

-------------------------------------------------------------------------------------------------

Having homepages that target the correct audience is an excellent way to manage your content. Sharing with the correct audiences allows even more control over valuable assets. For instance, if an asset is specific to a certain region, language, or country specific event, then the asset and correlating homepage should only be shown to a specific audience. 

How?[](https://support.bynder.com/hc/en-us/articles/28990876159634#h_01K3G98XA25TWPY28C4QWF1174)

-------------------------------------------------------------------------------------------------

### Activating Targeted Pages

1.  Start by **creating** or **editing** an existing homepage.
2.  In the **Inspector panel** to the righthand side of the editor, **turn on the “Targeted Pages” toggle**.  
    ![](https://support.bynder.com/hc/article_attachments/28991342486930)
3.  Once the toggle is activated, a **side panel** will appear on the left, showing:
    *   Your **default homepage** at the top (fallback)
    *   An option to **add new targeted pages** using the **“+” icon**
4.  You can either s**tart from a blank page,** **d****uplicate an existing homepage,** or **u****se a template.** **![](https://support.bynder.com/hc/article_attachments/28991342487954)**
5.  Users can now edit the page with sections and widgets just as they do with the default homepage. 

_Note:_ There must be a default homepage published at all times. Any user not part of a targeted audience will see the default homepage. All pages linked to the default homepage will have the same theme.

### Creating Targeting Rules

1.  Click the **“Manage Targeting”** button to assign your targeted pages to specific users or groups. You’ll see your default homepage and the targeted pages you've created.
2.  Under the **“Access”** column, assign users, groups, or profiles to each targeted page.

_Note:_ If a user belongs to multiple groups, they will see the page that appears higher in the list.
