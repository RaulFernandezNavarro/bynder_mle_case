# Understanding Preview Supported File Types – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013936359

Summary[](https://support.bynder.com/hc/en-us/articles/360013936359#h_01HRZ243Q6S18VFAETRBRJAR3D)

--------------------------------------------------------------------------------------------------

All file types can be uploaded and stored in your Bynder portal, however not all file types can be previewed. You can preview an asset when it has a maximum file size of 50 GB and one dimensions maximum pixel size is 25000 pixels. 

_Note: When uploading a new version of an asset the type cannot be changed. This means if you upload an image it should stay as an image. If the type is changed the preview thumbnail will be unavailable._

-----------------------------------------------------------------------------------------------

**Why can't I see a preview for my office files?**

The maximum file size that is supported for office files is 400MB. Office files larger than that can be uploaded, but a preview will not be generated.

**Are emojis supported when previewing Office files?**  
Yes, emojis are supported during preview, but they are rendered in greyscale using version 16 of the Noto font set. Any derivative documents generated from the original will also display the emojis in this greyscale style.

**Why does the font on my preview look different than the custom font of my file?**

Bynder cannot support _all_ fonts within preview, some custom fonts will therefore appear as a default font in preview. The file itself will continue to have the custom font. 

**Why is my file not giving me a preview when it is a supported file type?**

It could have been flagged by the antivirus feature in the Waiting Room. If that is not the case it could be that the file is password-protected. 

**I can't preview my Adobe Illustrator file?**

It is not preview-able if the file was not saved as PDF compatible.
