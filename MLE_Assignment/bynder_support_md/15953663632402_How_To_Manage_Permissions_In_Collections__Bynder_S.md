# How To Manage Permissions In Collections – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/15953663632402

You can control who can view or edit your Collections through Collection management. When you [share a collection](https://support.bynder.com/hc/en-us/articles/15953779016594)
, you can decide if the user can [view](https://support.bynder.com/hc/en-us/articles/15953663632402#h_01HNDXZ7ANF85XESS11PEPT9TJ)
, [download](https://support.bynder.com/hc/en-us/articles/15953663632402#h_01HNDXZJ11KCPG4VERSB7HPSZX)
, or [manage the assets](https://support.bynder.com/hc/en-us/articles/15953663632402#h_01HNDXZZDEDV8SD05ZCZ8T5P06)
 in a Collection. You can also [transfer Collection ownership](https://support.bynder.com/hc/en-us/articles/360013871300)
 to another user.

If you have permission, you can create Collections to be [published online](https://support.bynder.com/hc/en-us/articles/15954306339474)
.  

Your Collections tab shows the collections you shared or others shared with you. Learn more [here](https://support.bynder.com/hc/en-us/articles/15953981692434)
. 

**Who can manage collection permissions?**

How to Manage Collection Permission[](https://support.bynder.com/hc/en-us/articles/15953663632402#h_01HJ9BR01D0ZSN6KX944YJDFRA)

--------------------------------------------------------------------------------------------------------------------------------

1.  Follow the steps outlined [here to share a collection.](https://support.bynder.com/hc/en-us/articles/15953779016594)
    
    ![Screenshot 2023-12-22 at 1.16.59 PM.png](https://support.bynder.com/hc/article_attachments/15953663629970)
    
2.  Once you select **Share,** a pop-up window will appear where you can either [**Share with an individual**](https://support.bynder.com/hc/en-us/articles/15953663632402#h_01HNZZM6ZV2Q1X5HHFRNMWA139)
     or [**Create a Link**.](https://support.bynder.com/hc/en-us/articles/15953663632402#01HNZZMF8336G5RC3FEMTWGRW3)
    
    ![Screenshot 2024-02-06 at 2.53.31 PM.png](https://support.bynder.com/hc/article_attachments/16865344925586)
    

Share with Specific People[](https://support.bynder.com/hc/en-us/articles/15953663632402#01HNZZPN346XD0KGFP42F320WE)

---------------------------------------------------------------------------------------------------------------------

1.  Type in the user, group, or email address you want to share the collection with. 
    
    ![Screenshot 2024-07-10 at 3.25.40 PM.png](https://support.bynder.com/hc/article_attachments/20073202543762)
    
2.  Set their permissions to the collection. 
    *   **View Collection:** Allows users to view assets only.
        *   View collection can mean different things based on whether a logged-in user views the collection vs an external non-logged-in person. 
    *   **Download assets:** Allows users to download assets
    *   **Download all assets (including watermarked):** Allows users to download assets with a watermark.
        *    Disabling the profile permission **to download** [**watermarked assets**](https://support.bynder.com/hc/en-us/articles/18433246503442-Permissions-in-Asset-Bank#:~:text=Show%20all%20assets%20as%20watermarked%20%2D%20Displays%20all%20assets%20as%20watermarked%20(set%20it%20for%20specific%20user%20profiles).)
             will also remove the Download all assets (including watermarked) option in the dropdown. 
    *   **Manage Collection:** This can only be granted to Bynder users and allows them to [edit the collection.](https://support.bynder.com/hc/en-us/articles/15953663632402#h_01HNDXZZDEDV8SD05ZCZ8T5P06)
          
3.  **Uncheck**/ **Check** if you want to **require login** for users to access Collection.
    *   Uncheck this if you are sharing Collection with a non-Bynder user
4.  (Optional) Add message
5.  Uncheck/ Check **send email** if you want the recipient to receive an email once the Collection has been shared.
6.  (Optional) Indicate if you want to set a **Valid from/ Valid until.**
    
    ![Screenshot 2024-02-06 at 2.59.25 PM.png](https://support.bynder.com/hc/article_attachments/16865344932626)
    
7.  Select **Share.**

_Note: The users default permission profile may override the permission profile set here. This means that if a user has download rights overall they will be able to download assets even with a view only permission._

Share Link Settings[](https://support.bynder.com/hc/en-us/articles/15953663632402#01HNZZMF8336G5RC3FEMTWGRW3)

--------------------------------------------------------------------------------------------------------------

Before you share the generated link, make sure you set up the correct permissions.

1.  A link will automatically generate that you can share with Bynder and Non-Bynder users (depending on how access is set up).
2.  Select from the dropdown menu if you would like people with access to the link to be able to:
    
    ![Screenshot 2024-02-06 at 3.03.07 PM.png](https://support.bynder.com/hc/article_attachments/16865359274642)
    
    *   **View Collection:** Allows users to [View assets only.](https://support.bynder.com/hc/en-us/articles/15953663632402#h_01HNDXZ7ANF85XESS11PEPT9TJ)
        
    *   **Download assets:** [Allows users to download assets](https://support.bynder.com/hc/en-us/articles/15953663632402#h_01HNDXZJ11KCPG4VERSB7HPSZX)
        *   You can check the box to **Only allow downloads in low resolution.**
    *   **Download all assets (including watermarked):** Allows users to download watermarked assets.
        *   You can check the box to **Only allow downloads in low resolution.**
3.  Uncheck/ Check if you want to **require login** for users to access Collection.
    *   Uncheck this if you are sharing Collection with a non-Bynder user
4.  (Optional) Indicate if you want to set a **Valid from/ Valid until.**
5.  Click **Save** **Changes**.

_Note: The users default permission profile may override the permission profile set here. This means that if a user has download rights overall they will be able to download assets even with a view only permission._

**Why can't an external user preview a PDF shared with "view only" permission in a collection?**

External users cannot fully preview the document because a watermark is automatically applied, and the "Preview document" button is disabled to prevent potential downloads. While users may attempt to use the right-click "Save as" option, this will not download the full document. If you want to allow a user to download the asset you can either change the access rights to **download assets.**
