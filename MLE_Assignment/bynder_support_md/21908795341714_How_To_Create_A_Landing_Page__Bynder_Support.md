The Landing Page is an amazing feature of our CX User Community solution. Landing pages can be created for internal or external use, making it a highly flexible tool. It is easy to customize your landing page by adding content widgets such as text, buttons, cards, images, videos, collections, and more. Landing pages is an excellent way to make your homepage self-serviceable and engaging.

Who?[](https://support.bynder.com/hc/en-us/articles/21908795341714-How-To-Create-A-Landing-Page#h_01JMCYRAK3GF7E0Y8TSW47Z7SS)

------------------------------------------------------------------------------------------------------------------------------

_Note: This feature is automatically enabled when Brand Guidelines is enabled._

Once enabled Bynder Admins can assign _Creator_ or _Manager_ rights to users that should be able to create landing pages.

Why?[](https://support.bynder.com/hc/en-us/articles/21908795341714-How-To-Create-A-Landing-Page#h_01JMCYR6793KV2ZSES7G8Y1VZF)

------------------------------------------------------------------------------------------------------------------------------

Bynder uses highly customizable landing page creation to allow the user to have full flexibility in their creations. This works well when creating a landing page from scratch or when using one of Bynder's many scalable and in-depth templates. Templates are pre-structured, customizable, and designed for professional, on-brand content with minimal effort. Whichever way you choose to use landing pages, Bynder has the flexibility to make it work. 

How?[](https://support.bynder.com/hc/en-us/articles/21908795341714-How-To-Create-A-Landing-Page#h_01JMCYR1RY8G24V3FBNWT9B9JV)

------------------------------------------------------------------------------------------------------------------------------

### How To Create A New Landing Page

1\. From your Bynder portal select **CX for User Community** section.

_Note: This may be a different name in some portals._

2\. In the left sidebar, click **Landing** **Pages**.

![](https://support.bynder.com/hc/article_attachments/21909886334482)

3\. Click the **＋New Landing Page** located in the top right corner.

![](https://support.bynder.com/hc/article_attachments/21909900798994)

4\. Enter a title for the new page in the **Title** **field**.

_Note: You can change the title of your landing page by selecting ![Screenshot](https://support.bynder.com/hc/article_attachments/21909886340754) to edit the page title and click ![Screenshot](https://support.bynder.com/hc/article_attachments/21909900805138)to save it._

![](https://support.bynder.com/hc/article_attachments/21909900806674)

5\. Select your desired page layout:

*   *   **Blank** **Page**: Start from scratch.
    *   **Page** **Template**: Aids in the creation process by providing a pre-built layout.
        *   If enabled users have the option to select a theme directly from the template library, allowing for branded templates to be customized at the click of a button. 
            
            ![](https://support.bynder.com/hc/article_attachments/21909886345362)
            

6\. Select **Create** to generate the new page. You can now begin to add content to your landing page.![](https://support.bynder.com/hc/article_attachments/21909886346130)

### How To Set The _Look And Feel_

You can set the **look and feel** of your landing page by applying the right theme in the **Settings** section.

1.  Navigate to your Bynder portal.
2.  Select **CX for User Community** > **Landing pages.**
3.  Add a new landing page or open an existing one. 
4.  Select **Edit** > **Settings**. **![](https://support.bynder.com/hc/article_attachments/21909886346642)**
5.  Select **Theme** dropdown and select the theme you want to apply.

### How to Add Content to your Landing Page

1.  If you are starting from scratch click in the blank area at the top of the page and select the white space.![](https://support.bynder.com/hc/article_attachments/21909900811538)
2.  Click the ![](https://support.bynder.com/hc/article_attachments/21909900812562)to start adding content.

Alternatively, if you are starting from a **template** you can select the pre-existing sections and begin modifying the content.![](https://support.bynder.com/hc/article_attachments/21909886350354)  
  
 

1.  Ensure you're in the **Section** tab of the right sidebar to make the necessary customizations.
2.  Select the section you want to customize.**![GIF-templates.gif](https://support.bynder.com/hc/article_attachments/21909886351378)**
3.  Select the![](https://support.bynder.com/hc/article_attachments/21909900815250)button inside a section to add content types from the  **media**, **formatting** or **embed** options.**![](https://support.bynder.com/hc/article_attachments/21909900816530)**
