# CX for User Community Packages – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/21909413766546-CX-for-User-Community-Packages

Three package options are available in Content Experiences (CX) for User Community. Each option provides flexibility and scalability, allowing you to enhance user engagement by delivering targeted, branded content through Guides and Landing Pages. 

If you're already using Brand Guidelines, you'll automatically have access to these features. If you do not currently have Brand Guidelines (CX for User Community) enabled, please reach out to your Customer Success Contact to learn how to enable this solution and discuss any associated costs.

Basic Package[](https://support.bynder.com/hc/en-us/articles/21909413766546-CX-for-User-Community-Packages#h_01J9W1BCZJ53JVMGQ0B8HNV6QA)

-----------------------------------------------------------------------------------------------------------------------------------------

The **Basic** package is ideal for small teams or organizations looking to create and share a limited number of guides and landing pages, typically the core guidelines and a single landing page to help users navigate to the right content.

*   **Theming**: Contact your Customer Success Contact to learn more.
*   **Unlimited** **Pages in Guides**: Guides can include unlimited pages
*   **User Access Levels:** Offers **two** access levels to control who can view the content:
    *   All Licensed Users
    *   Public Access 
*   **Custom Links:** 

Advanced Package[](https://support.bynder.com/hc/en-us/articles/21909413766546-CX-for-User-Community-Packages#h_01J9W1CARVXVT7QK8WDVPHB2P9)

--------------------------------------------------------------------------------------------------------------------------------------------

The **Advanced** **package** expands the number of available landing pages and guides, and access to include external users, allowing organizations to create and deliver content tailored to a more diverse set of use cases and audiences 

*   **Landing Pages: Contact your Customer Success Contact to learn more.**
*   **Theming**: Contact your Customer Success Contact to learn more.
*   **Unlimited** **Pages** **in** **Guides**: Guides can include unlimited pages
*   **User** **Access** **Levels**: Offers **four** access levels:
    *   All Licensed Users
    *   Public Access
    *   Specific Users (users can be selected individually)
    *   Allow access and sharing control within guides at the page and chapter levels.

Enterprise Package[](https://support.bynder.com/hc/en-us/articles/21909413766546-CX-for-User-Community-Packages#h_01J9W1FA0SZ92Y0D10REG74XFY)

----------------------------------------------------------------------------------------------------------------------------------------------

For larger organizations, the **Enterprise** package provides the most flexibility and control over content experiences for large organizations with complex content architectures, multiple audiences, and the need for comprehensive access control.

*   **Landing Pages**: Contact your Customer Success Contact to learn more.
*   **Theming**: Contact your Customer Success Contact to learn more..
*   **Unlimited Pages in Guides:** Guides can include unlimited pages.
*   **Guide Groups:** Allows guides to be grouped for easier navigation and organization. 
*   **User Access Levels:** Offers **five** levels of access:
    *   All Licensed Users
    *   Public Access
    *   Specific Users (users can be selected individually)
    *   Allow access and sharing control within guides at the page and chapter levels.
    *   Allow access and sharing control at the guide group level.
