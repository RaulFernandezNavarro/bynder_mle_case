# Understanding Supported File Types For Conversion And Derivative Creation – Bynder Support

**Source URL:** https://support.bynder.com/hc/en-us/articles/360013875220

Summary[](https://support.bynder.com/hc/en-us/articles/360013875220#h_01JKTM72RZN6FQJ2XJBVGQEE8B)

--------------------------------------------------------------------------------------------------

Bynder provides the flexibility to convert various file types into different formats for creating derivatives, as well as for previewing, and downloading. Below we summarize the supported file types and the possible conversion options. 

### Image

|     | JPG | PNG | TIFF |
| --- | --- | --- | --- |
| **JPG/JPEG ->** | ✅   | ✅   | ✅   |
| **PNG ->** | ✅   | ✅   | ✅   |
| **GIF ->** | ✅   | ✅   | ✅   |
| **TIFF ->** | ✅   | ✅   | ✅   |
| **BMP ->** | ✅   | ✅   | ✅   |
| **INDD ->** | Preview only (embedded thumbnail) | ❌   | ❌   |
| **SVG ->** | ✅   | ✅   | ✅   |
| **AI** (non-PDF) **\->** | ❌   | ❌   | ❌   |
| **AI** (PDF compatible) **\->** | ✅   | ✅   | ✅   |
| **PSD/PSB ->** | ✅   | ✅   | ✅   |
| **PDF ->** | ✅   | ✅   | ✅   |
| **EPS ->** | ✅   | ✅   | ✅   |
| **HEIC ->** | ✅   | ✅   | ✅   |
| **AVIF ->** | ✅   | ✅   | ✅   |

_Note: Assets cannot be converted to: .GIF, .BMP, .INDD, .SVG, .AI, .PSD, .PSB, .PDF, .WEBP, or .EPS_

### Video

Supported Video file types are: .WMV, .MPEG, .MOV, .AVI, .FLV, .VOB, .MKV, .MV4, .F4V, .3GPP, .3GP, .OGV, .TS, .MTS, .M2TS, .3G2, .M2V, .WEBM

All supported video file types can be converted to either: .mp4 with h264 video codec and aac audio coded, or .WebM with VP9 video codec and Ogg Vorbis audio codec

### Audio

Audio file types are supported for previewing, using, and downloading. However, derivatives are not available for audio files.  
The following are supported audio file format types: .WAV, .MP3, .OCN, .WMA, .M4A, .OGG, .AIFF, .AIF, and .M4R.

### Document

Documents of type: .DOC, .DOCX, .PPT, and .PPTX can be converted _only_ to type .PDF. 

Note: Bynder does not recommend attempting derivatives with documents.
